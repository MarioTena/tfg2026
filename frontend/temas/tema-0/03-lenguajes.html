Lenguajes de programación

Diferentes lenguajes, mismo objetivo.

Existen muchos lenguajes porque existen muchos tipos de problemas.

  • Lenguajes fáciles de leer
  • Lenguajes cercanos al hardware
  • Lenguajes para aprender
  • Lenguajes para sistemas críticos
Aprender a programar ≠ aprender un lenguaje
