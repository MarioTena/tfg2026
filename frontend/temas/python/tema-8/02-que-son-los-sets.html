Qué es un set

Un set es una colección de elementos únicos. Su característica más importante es que no admite duplicados. 

numeros = {1, 2, 3, 4}
letras = {"a", "b", "c"}

No debes pensarlo como una secuencia

A diferencia de listas, strings o tuplas, el set no es una estructura para trabajar por posición. Aquí no tiene sentido preguntar “qué hay en el índice 0”.

El set sirve mejor para preguntar “¿está este elemento?” que para preguntar “¿qué hay en esta posición?”.

Qué ocurre con duplicados

valores = {1, 2, 2, 3, 3, 3}
print(valores)

El resultado solo conserva una copia de cada valor.

Lista frente a set

Lista

Mantiene orden y puede tener repetidos.

Set

No está pensado para orden y elimina duplicados.

Lista

Sirve bien si importa la posición.

Set

Sirve bien si importa la unicidad y la pertenencia.

Convertir una lista en set

numeros = [1, 2, 2, 3, 3, 4]
unicos = set(numeros)
print(unicos)

Es una forma muy útil de limpiar duplicados.

Crear un set vacío

conjunto = set()

No debes usar {} para esto, porque eso crea un diccionario vacío.

Añadir y eliminar elementos

add()

colores = {"rojo", "verde"}
colores.add("azul")

Añade un nuevo elemento.

remove()

colores = {"rojo", "verde", "azul"}
colores.remove("verde")

Elimina un elemento existente.

Comprobar pertenencia

numeros = {1, 2, 3, 4}
print(3 in numeros)
print(10 in numeros)

La pertenencia es una de las ideas centrales del set.

Errores frecuentes

Error 1

Querer usar índices como si fuera una lista.

Error 2

Interpretar el orden al imprimir como si fuese estable.

Error 3

Crear {} pensando que es un set vacío.

Error 4

Olvidar que los duplicados desaparecen automáticamente.
