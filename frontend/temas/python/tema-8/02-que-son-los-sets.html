8.2 · Qué son los sets: colecciones sin duplicados

¿Qué es un set?

Un set es una colección no ordenada de elementos únicos. Su característica más importante es que no admite duplicados. 

numeros = {1, 2, 3, 4}
letras = {"a", "b", "c"}

Qué ocurre con duplicados

valores = {1, 2, 2, 3, 3, 3}
print(valores)

El resultado solo conserva una copia de cada valor.

Un set es muy útil para eliminar repetidos de forma sencilla.

Diferencia con una lista

  • la lista mantiene orden y duplicados
  • el set no garantiza orden y elimina duplicados

Crear un set vacío

conjunto = set()

No debes usar {} para esto, porque eso crea un diccionario vacío.

Añadir elementos con add()

colores = {"rojo", "verde"}
colores.add("azul")
print(colores)

Eliminar elementos

colores = {"rojo", "verde", "azul"}
colores.remove("verde")
print(colores)

Comprobar pertenencia

numeros = {1, 2, 3, 4}
print(3 in numeros)
print(10 in numeros)

Los sets son especialmente buenos para comprobaciones de pertenencia rápidas.

💭 Piensa antes de ejecutar

datos = {1, 2, 2, 3}
datos.add(4)
print(datos)

Errores frecuentes

  • pensar que un set mantiene un orden fijo
  • esperar ver duplicados
  • crear un set vacío con {}

Mini-reto guiado

Crea un set con varios números repetidos y muestra el resultado.

numeros = {1, 1, 2, 2, 3, 4, 4}
print(numeros)
