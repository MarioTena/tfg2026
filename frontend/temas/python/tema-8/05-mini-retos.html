Qué entrenas aquí

Tuplas Sets Diseño

Objetivo

Vas a practicar cómo usar tuplas y sets y, sobre todo, cuándo conviene elegir una estructura u otra.

Idea clave: una estructura buena no es la más conocida, sino la que mejor encaja con el problema.

Cómo usar esta pantalla

  • lee el reto
  • identifica si importa orden, repetición o cambio
  • decide si usar tupla, set o incluso otra estructura
  • pruébalo en el playground y justifica tu elección
No te quedes solo en “funciona”. Intenta explicar por qué esa estructura representa mejor el caso.

Misiones rápidas

Mini-reto 1 · Coordenada

Usa una tupla para representar datos fijos.

punto = (4, 9)
print(punto[0])
print(punto[1])

Mini-reto 2 · Eliminar duplicados

Convierte una lista en set para quedarte con valores únicos.

numeros = [1, 2, 2, 3, 4, 4, 5]
unicos = set(numeros)
print(unicos)

Mini-reto 3 · Elementos comunes

Trabaja con la intersección entre sets.

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}
print(a & b)

Mini-reto 4 · Añadir a set

Modifica un conjunto sin duplicados.

tags = {"python", "html"}
tags.add("css")
print(tags)

Mini-reto 5 · Unión de conjuntos

Combina dos sets para obtener todos los elementos distintos.

grupo_a = {"Ana", "Luis"}
grupo_b = {"Luis", "Marta"}

todos = grupo_a | grupo_b
print(todos)

Mini-reto 6 · Elegir estructura

Decide si usarías lista, tupla o set para:

  • asientos numerados
  • latitud y longitud
  • usuarios únicos conectados

Cómo pensar este tipo de ejercicios

Antes de escribir, pregúntate qué propiedad manda en el problema.

Paso a paso

  1. piensa si el orden importa
  2. piensa si puede haber duplicados
  3. piensa si el valor debería cambiar o no
  4. elige estructura antes de escribir código
  5. justifica por qué esa estructura encaja mejor

Regla útil

Si los datos son fijos, una tupla suele tener sentido. Si quieres unicidad, un set suele ser mejor opción.

Pregunta típica de entrevista

Pregunta: ¿Cuándo usarías un set en lugar de una lista?

Respuesta sólida: cuando quiero evitar duplicados o hacer operaciones de pertenencia y conjunto de manera más natural.
