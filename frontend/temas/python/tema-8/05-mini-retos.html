Qué entrenas aquí

Aquí no solo practicas sintaxis. Aprendes a decidir mejor qué estructura encaja con cada situación y por qué.

Comparación Criterio Práctica guiada

Objetivo

Vas a practicar cómo usar tuplas y sets y, sobre todo, cómo justificar su elección frente a una lista u otra alternativa.

Idea clave: una estructura buena no es la más conocida, sino la que representa mejor el caso real.

Cómo usar esta pantalla

  • lee la situación o el código
  • identifica si importa orden, repetición, pertenencia o cambio
  • decide si usarías lista, tupla o set
  • intenta justificarlo con una frase clara
No te quedes en “funciona”. Intenta explicar por qué esa estructura modela mejor el problema.

Reto 5 · Dirección de la diferencia

Código

b = {3, 4, 5}

print(a - b)
print(b - a)

Qué entrenas

¿Por qué no devuelven lo mismo? Explica qué pregunta responde cada una.

La diferencia es una de las operaciones donde más se nota que el orden de los operandos sí importa.

Reto 8 · Qué harías con este caso

Situación

Quieres saber qué alumnos aparecen en dos listas de asistencia distintas sin repetir nombres.

Prueba obligatoria

¿Qué estructura y qué operación usarías? Explica por qué el enfoque encaja con la pregunta.

Método rápido para elegir estructura

Qué hacer antes de decidir entre lista, tupla o set.

Paso a paso

  1. pregunta si importa el orden
  2. pregunta si puede haber duplicados
  3. pregunta si el grupo debería cambiar
  4. pregunta si te importa la pertenencia más que la posición
  5. elige la estructura que mejor responda a esas condiciones

Regla útil

Si una estructura “sirve” pero obliga a pensar el problema de forma menos natural, probablemente no sea la mejor.
