8.5 · Mini retos: integrar tuplas y sets

Mini-reto 1 · Coordenada

punto = (4, 9)
print(punto[0])
print(punto[1])

Mini-reto 2 · Eliminar duplicados

numeros = [1, 2, 2, 3, 4, 4, 5]
unicos = set(numeros)
print(unicos)

Mini-reto 3 · Elementos comunes

a = {1, 2, 3, 4}
b = {3, 4, 5, 6}
print(a & b)

Mini-reto 4 · Añadir a set

tags = {"python", "html"}
tags.add("css")
print(tags)

Mini-reto 5 · Elegir estructura

Decide si usarías lista, tupla o set para:

  • asientos numerados
  • latitud y longitud
  • usuarios únicos conectados
