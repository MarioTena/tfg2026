Una tupla es una secuencia ordenada, parecida a una lista, pero pensada para datos que no deben modificarse.

Objetivo Orden + estabilidad

Qué problema resuelve

A veces necesitas guardar varios datos juntos, manteniendo el orden, pero dejando claro que no deberían cambiar durante la ejecución del programa.

Coordenadas

Fechas

Datos fijos

Funciones

Idea clave: una tupla se parece a una lista, pero comunica que sus datos no deberían cambiar.

Qué debes vigilar

  • una tupla usa paréntesis ()
  • mantiene el orden de los elementos
  • permite indexación y slicing
  • no se puede modificar directamente
Piensa así:
una lista suele representar datos flexibles; una tupla, datos ordenados que quieres mantener estables.

Qué es una tupla

Una tupla es una estructura de datos ordenada, parecida a una lista, pero con una diferencia fundamental: no se puede modificar después de crearla. 

coordenada = (10, 20)
colores = ("rojo", "verde", "azul")

Tupla vs lista

Ambas

Son secuencias ordenadas.

Ambas

Permiten indexación y slicing.

Lista

Es mutable y se puede cambiar.

Tupla

Es inmutable y no se modifica directamente.

Crear tuplas

numeros = (1, 2, 3)
datos = ("Ana", 30, True)

También existe la tupla de un solo elemento, pero necesita una coma. 

una_tupla = (5,)
Sin la coma, Python no lo interpreta como tupla de un elemento.

Acceder a sus elementos

colores = ("rojo", "verde", "azul")
print(colores[0])
print(colores[-1])

Igual que en listas y strings, puedes acceder por índice positivo o negativo.

Cuándo usar tuplas

Coordenadas

(x, y) o (latitud, longitud).

Fechas simples

Un grupo fijo de valores relacionados.

Datos que no cambian

Cuando quieres transmitir estabilidad.

Varios retornos

Al devolver varios resultados desde una función.

Error típico: intentar modificar una tupla

colores = ("rojo", "verde", "azul")
colores[1] = "amarillo"

Esto da error porque la tupla es inmutable.

Cómo pensar una tupla antes de usarla

Paso 1

Pregúntate si el orden importa.

Paso 2

Pregúntate si esos datos deberían cambiar.

Paso 3

Si el grupo es fijo, una tupla suele encajar bien.

Paso 4

Si necesitas cambios frecuentes, probablemente quieras una lista.

Piensa antes de ejecutar

Antes de lanzar el código, piensa qué representa cada posición y por qué una tupla encaja en este caso.

Código

punto = (3, 7)
print(punto[0])
print(len(punto))

Preguntas

  • ¿qué devuelve punto[0]?
  • ¿qué representa len(punto)?

Errores frecuentes

Error 1

Confundir una tupla con una lista.

Error 2

Olvidar la coma en una tupla de un solo elemento.

Error 3

Intentar modificar un valor de la tupla.

Error 4

Usar tupla cuando en realidad necesitas una estructura mutable.

Buenas prácticas

Usa tupla si es fijo

Te ayuda a comunicar intención en el código.

No la fuerces

Si vas a modificar datos, usa lista.

Nombra bien

Variables como coordenada o fecha ayudan a entender la estructura.

Piensa en estabilidad

La tupla encaja bien cuando el grupo de valores representa algo estable.
