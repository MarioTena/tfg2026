8.1 · Qué son las tuplas: secuencias ordenadas e inmutables

¿Qué es una tupla?

Una tupla es una estructura de datos ordenada, parecida a una lista, pero con una diferencia fundamental: no se puede modificar después de crearla. 

coordenada = (10, 20)
colores = ("rojo", "verde", "azul")
Una tupla es útil cuando quieres dejar claro que esos datos no deben cambiar.

Tupla vs lista

  • ambas son secuencias ordenadas
  • ambas permiten indexación y slicing
  • la lista es mutable
  • la tupla es inmutable

Crear tuplas

numeros = (1, 2, 3)
datos = ("Ana", 30, True)

También existe la tupla de un solo elemento, pero necesita una coma. 

una_tupla = (5,)

Acceder a sus elementos

colores = ("rojo", "verde", "azul")
print(colores[0])
print(colores[-1])

⚠️ Error típico: intentar modificar una tupla

colores = ("rojo", "verde", "azul")
colores[1] = "amarillo"

Esto da error porque la tupla es inmutable.

Cuándo usar tuplas

  • coordenadas
  • fechas simples
  • datos fijos que no deben cambiar
  • devolver varios valores desde una función

💭 Piensa antes de ejecutar

punto = (3, 7)
print(punto[0])
print(len(punto))

Mini-reto guiado

Crea una tupla con día, mes y año y muestra cada valor.

fecha = (14, 3, 2026)
print(fecha[0])
print(fecha[1])
print(fecha[2])
