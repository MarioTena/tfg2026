2.2 · Comparaciones y booleanos: la base matemática de las decisiones

¿Qué es una comparación?

Una comparación es una expresión que enfrenta dos valores para responder una pregunta exacta. El resultado siempre es un booleano: True o False. 

print(5 > 3)
print(10 == 10)
print(7 < 2)

En programación, comparar significa convertir una pregunta en una respuesta formal.

¿Qué es un booleano?

Un booleano es un tipo de dato que solo puede tener dos valores: True o False. Es el tipo que usa el programa para decidir. 

aprobado = True
bloqueado = False
print(type(aprobado))
Los condicionales no entienden “más o menos”. Entienden verdadero o falso.

Operadores de comparación

  • == igual que
  • != distinto de
  • > mayor que
  • < menor que
  • >= mayor o igual que
  • <= menor o igual que
print(8 == 8)
print(8 != 3)
print(9 > 2)
print(4 < 1)
print(7 >= 7)
print(2 <= 10)

⚠️ El error más importante: = no es ==

Este error es central. Si no lo entiendes bien, construirás mal casi todo lo que venga después.

  • = significa asignar
  • == significa comparar
edad = 18          # asignación
print(edad == 18)  # comparación

La primera línea guarda un valor. La segunda pregunta si el valor guardado es igual a 18.

Comparar números

precio = 50
print(precio > 100)
print(precio <= 50)
print(precio == 50)
print(precio != 30)

Las comparaciones numéricas son la base de los rangos, las validaciones y la lógica de negocio.

Comparar textos

usuario = "ana"
print(usuario == "ana")
print(usuario == "Ana")
print(usuario != "admin")

Python compara exactamente el contenido del texto. Las mayúsculas y minúsculas cuentan.

"ana" y "Ana" no son el mismo valor.

Guardar el resultado de una comparación

Una comparación se puede guardar en una variable. Esto es muy útil para hacer el código más legible. 

edad = 17
es_mayor = edad >= 18
print(es_mayor)

Aquí es_mayor vale False.

Comparaciones dentro de condicionales

nota = 6

if nota >= 5:
    print("Has aprobado")
else:
    print("Has suspendido")

El if no evalúa números directamente: evalúa el resultado booleano de la comparación.

Comparaciones encadenadas

Python permite escribir rangos de forma muy natural. 

edad = 20
print(18 <= edad <= 30)

Esto equivale a decir: edad >= 18 and edad <= 30, pero es más limpio.

Verdad y falsedad de valores

Además de las comparaciones, Python puede interpretar algunos valores como verdaderos o falsos. Esto se conoce como truthiness. 

print(bool(0))
print(bool(1))
print(bool(""))
print(bool("Hola"))
  • 0 suele interpretarse como falso
  • un número distinto de 0 suele interpretarse como verdadero
  • una cadena vacía suele interpretarse como falsa
  • una cadena con contenido suele interpretarse como verdadera

No necesitas dominarlo todo ahora, pero es importante empezar a verlo.

Comparar booleanos

activo = True
print(activo == True)
print(activo == False)

Aunque se puede hacer, normalmente es más claro usar la variable directamente en condiciones.

💭 Piensa antes de ejecutar

numero = 10
print(numero == 10)
print(numero != 10)
print(numero > 5)
print(numero < 3)
  • ¿Qué expresiones dan True?
  • ¿Cuál da False?
  • ¿Qué cambiaría si numero fuera 3?

Casos límite

Los casos límite son valores que están justo en la frontera de una regla. Son fundamentales al programar bien. 

edad = 18
if edad >= 18:
    print("Mayor de edad")

Aquí 18 es un caso límite. Debes pensar siempre en estos valores frontera.

Errores frecuentes

  • confundir asignación con comparación
  • comparar texto sin tener en cuenta mayúsculas
  • olvidar que el resultado de toda comparación es booleano
  • no probar casos límite
  • usar comparaciones innecesariamente complejas

Mini-reto guiado

Guarda dos números y muestra el resultado de estas preguntas:

  • ¿son iguales?
  • ¿el primero es mayor?
  • ¿el segundo es mayor o igual?
a = 8
b = 12

print(a == b)
print(a > b)
print(b >= a)
