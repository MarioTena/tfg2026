2.1 · Condicionales: enseñar al programa a decidir

¿Qué es un condicional?

Un condicional es una estructura que permite que el programa elija entre distintos caminos. Gracias a los condicionales, el programa puede hacer una cosa si se cumple una condición y otra distinta si no se cumple. 

edad = 20

if edad >= 18:
    print("Puedes entrar")

Aquí el programa se hace una pregunta: ¿edad es mayor o igual que 18? Si la respuesta es True, ejecuta el bloque. Si la respuesta es False, lo salta.

Programar con condicionales es convertir reglas humanas en reglas exactas para una máquina.

La estructura básica de un if

if condicion:
    instruccion

Todo if tiene estas piezas:

  • la palabra clave if
  • una condición
  • dos puntos :
  • un bloque indentado
temperatura = 33

if temperatura > 30:
    print("Hace calor")

La indentación en Python es obligatoria

En muchos lenguajes se usan llaves para definir bloques. En Python, el bloque se define por la indentación. Eso significa que los espacios al inicio de la línea tienen significado. 

saldo = 100

if saldo > 0:
    print("Saldo disponible")
    print("Puedes operar")

print("Fin")

Las dos líneas indentadas pertenecen al if. La última ya está fuera del bloque.

⚠️ Error típico: dos puntos e indentación

edad = 20
if edad >= 18
print("Mayor de edad")

Este código es incorrecto porque:

  • falta el :
  • el bloque no está indentado
edad = 20
if edad >= 18:
    print("Mayor de edad")

if + else: dos caminos posibles

Cuando quieres cubrir dos posibilidades opuestas, usas if y else. 

edad = 16

if edad >= 18:
    print("Puedes votar")
else:
    print("No puedes votar")

Aquí siempre ocurrirá una de las dos opciones. Nunca ambas.

elif: varias decisiones encadenadas

elif sirve para comprobar condiciones adicionales cuando la anterior no se ha cumplido. Es útil cuando hay más de dos posibilidades. 

nota = 7

if nota >= 9:
    print("Sobresaliente")
elif nota >= 7:
    print("Notable")
elif nota >= 5:
    print("Aprobado")
else:
    print("Suspenso")

El orden importa. Python evalúa de arriba hacia abajo y se detiene en la primera condición verdadera.

El orden de las condiciones importa mucho

Observa este ejemplo incorrecto: 

nota = 9

if nota >= 5:
    print("Aprobado")
elif nota >= 7:
    print("Notable")
elif nota >= 9:
    print("Sobresaliente")

Aunque la nota sea 9, imprimirá Aprobado, porque la primera condición ya es verdadera. Las demás ni siquiera se revisan.

La versión correcta es empezar por el caso más restrictivo o más alto.

Razonamiento correcto de rangos

edad = 15

if edad < 12:
    print("Niño")
elif edad < 18:
    print("Adolescente")
else:
    print("Adulto")

Este código funciona porque si llega al segundo bloque, ya sabemos que edad no era menor que 12. No hace falta repetirlo.

Una buena condición no repite información innecesaria.

Condicionales con texto

rol = "admin"

if rol == "admin":
    print("Acceso completo")
else:
    print("Acceso limitado")

Las cadenas de texto se comparan carácter por carácter. Python diferencia mayúsculas y minúsculas. 

print("admin" == "Admin")   # False

Condicionales con booleanos

sesion_iniciada = True

if sesion_iniciada:
    print("Bienvenido")
else:
    print("Debes iniciar sesión")

Si una variable ya contiene True o False, puedes usarla directamente como condición. 

# Menos recomendable
if sesion_iniciada == True:
    print("Bienvenido")

Condicionales anidados

Un condicional anidado es un if dentro de otro if. Se usa cuando una decisión depende de otra previa. 

edad = 20
tiene_entrada = True

if edad >= 18:
    if tiene_entrada:
        print("Puedes pasar")
    else:
        print("Eres mayor de edad, pero no tienes entrada")
else:
    print("No eres mayor de edad")

Aunque esto funciona, no siempre es la forma más clara. Más adelante aprenderás a simplificar lógica combinando condiciones.

Errores conceptuales muy comunes

  • confundir = con ==
  • escribir mal el orden de los rangos
  • olvidar que Python para en la primera condición verdadera
  • pensar que dos if seguidos equivalen a if / elif
  • usar variables sin haber pensado todos los casos posibles

if y elif no siempre son iguales que varios if

Observa esta diferencia: 

nota = 9

if nota >= 5:
    print("Aprobado")
if nota >= 9:
    print("Sobresaliente")

Aquí se imprimirán dos mensajes. En cambio, con if / elif, solo uno. 

nota = 9

if nota >= 9:
    print("Sobresaliente")
elif nota >= 5:
    print("Aprobado")

💭 Piensa antes de ejecutar

numero = 0

if numero > 0:
    print("Positivo")
elif numero < 0:
    print("Negativo")
else:
    print("Cero")
  • ¿Qué muestra con 0?
  • ¿Qué muestra con 12?
  • ¿Qué muestra con -8?
  • ¿Por qué solo se imprime un mensaje?

Buenas prácticas al escribir condicionales

  • usa nombres claros: edad, tiene_permiso, es_admin
  • ordena de forma lógica los casos
  • evita repetir condiciones innecesarias
  • prueba valores límite: 0, 1, 17, 18, 65...
  • si la lógica se complica mucho, descompón el problema

Mini-reto guiado

Crea un programa que clasifique una temperatura:

  • menor que 10 → hace frío
  • entre 10 y 24 → temperatura agradable
  • 25 o más → hace calor
temperatura = 18

if temperatura < 10:
    print("Hace frío")
elif temperatura < 25:
    print("Temperatura agradable")
else:
    print("Hace calor")
