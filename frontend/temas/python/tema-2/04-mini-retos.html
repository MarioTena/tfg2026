Qué entrenas aquí

En este bloque no solo escribes condiciones. Aprendes a leerlas, prever el resultado, corregir errores y comprobar los valores donde más suele romperse la lógica.

Lectura Depuración Práctica guiada

Objetivo

Estos retos te ayudan a entender por qué un condicional entra en un bloque y no en otro, y a detectar errores antes de que se conviertan en malos hábitos.

Idea clave: un condicional correcto no solo ejecuta algo. También se entiende y se puede justificar.

Cómo usar esta pantalla

  • lee el código antes de ejecutarlo
  • intenta adivinar la salida
  • detecta si hay error de sintaxis o de lógica
  • prueba el valor frontera cuando exista
Si no puedes explicar por qué sale un resultado, todavía no controlas del todo la condición.

Reto 1 · ¿Qué imprime?

Código

edad = 18

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

Qué debes pensar

¿El 18 entra o no entra en la condición? Ese es el caso frontera.

Reto 2 · Detecta el error

Código incorrecto

nota = 7

if nota >= 5
    print("Aprobado")

Pregunta

¿Qué falta exactamente para que ese código funcione?

No basta con decir “está mal”. Debes identificar por qué está mal.

Reto 3 · Indentación

Código incorrecto

saldo = 100

if saldo > 0:
print("Saldo positivo")

Pregunta

¿Qué línea debería estar dentro del bloque? ¿Cómo debe quedar escrita?

Reto 4 · Orden de casos

Código

nota = 9

if nota >= 5:
    print("Aprobado")
elif nota >= 7:
    print("Notable")
elif nota >= 9:
    print("Sobresaliente")

Pregunta

¿Qué imprime realmente? ¿Por qué no llega al último caso?

Un condicional puede estar bien escrito y aun así estar mal planteado.

Reto 5 · Positivo, negativo o cero

Código

numero = 0

if numero > 0:
    print("Positivo")
elif numero < 0:
    print("Negativo")
else:
    print("Cero")

Qué entrenas

Comprobar si la lógica cubre todos los casos, incluido el valor frontera más importante: el 0.

Reto 6 · Comparación con texto

Código

usuario = "Ana"

if usuario == "ana":
    print("Acceso permitido")
else:
    print("Acceso denegado")

Pregunta

¿Qué imprime? ¿Cuál es el detalle que hace que falle la comparación?

Reto 7 · Lógica con or

Código

edad = 22
es_estudiante = False

if edad < 25 or es_estudiante:
    print("Tiene descuento")
else:
    print("No tiene descuento")

Pregunta

¿Por qué entra en el bloque si solo una parte es verdadera?

Reto 8 · Input y frontera

Código

edad = int(input("Edad: "))

if edad >= 18:
    print("Puede entrar")
else:
    print("No puede entrar")

Prueba obligatoria

Ejecuta el programa con 17, 18 y 19. No te quedes con una sola prueba.

Método rápido para probar un condicional

Qué hacer antes de dar un ejercicio por terminado.

Paso a paso

  1. identifica qué pregunta responde el programa
  2. localiza el valor frontera
  3. prueba un caso por debajo
  4. prueba el valor exacto
  5. prueba un caso por encima

Regla útil

Si solo pruebas un valor cómodo, puedes pensar que el programa funciona cuando todavía falla justo en el límite.
