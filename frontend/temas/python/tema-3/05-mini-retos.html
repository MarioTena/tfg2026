Qué entrenas aquí

En este bloque no solo escribes bucles. Aprendes a leerlos, seguir la evolución de las variables, detectar errores y entender por qué terminan o por qué podrían no terminar.

Lectura Depuración Práctica guiada

Objetivo

Estos retos te ayudan a dejar de ver los bucles como “algo que repite” y empezar a verlos como una lógica con inicio, evolución y final.

Idea clave: un bucle bien entendido no solo funciona. También sabes explicar qué pasa en cada vuelta.

Cómo usar esta pantalla

  • lee el código antes de ejecutarlo
  • intenta prever la salida
  • sigue mentalmente la variable que cambia
  • detecta si hay error de lógica o riesgo de bucle infinito
Si no puedes explicar cuántas iteraciones habrá o por qué termina, todavía no controlas del todo el bucle.

Reto 1 · ¿Qué imprime este for?

Código

for numero in range(1, 4):
    print(numero)

Qué debes pensar

¿Qué valores toma numero? ¿Cuántas iteraciones hay? ¿Se imprime el 4 o no?

Reto 2 · Límite de range()

Código

for numero in range(2, 6):
    print(numero)

Pregunta

¿Por qué se imprime hasta 5 y no hasta 6? ¿Qué valor queda fuera del rango?

En range(), entender el límite superior es más importante de lo que parece.

Reto 3 · Acumulador paso a paso

Código

suma = 0

for numero in range(1, 4):
    suma = suma + numero

print(suma)

Qué debes pensar

¿Cuánto vale suma después de cada iteración? ¿Qué valor final imprime?

Reto 4 · Detecta el bucle infinito

Código incorrecto

contador = 1

while contador <= 5:
    print(contador)

Pregunta

¿Por qué este bucle no termina? ¿Qué línea falta exactamente para que funcione bien?

En while, el error más común no es la sintaxis: es olvidar qué debe cambiar en cada vuelta.

Reto 5 · While con condición

Código

contador = 2

while contador <= 8:
    print(contador)
    contador = contador + 2

Qué entrenas

Seguir mentalmente una variable de control y entender cuándo deja de cumplirse la condición.

Reto 6 · break

Código

for letra in "programacion":
    if letra == "g":
        break
    print(letra)

Pregunta

¿Qué letras se imprimen? ¿Por qué el bucle se detiene justo en ese punto?

Reto 7 · continue

Código

for numero in range(1, 6):
    if numero == 3:
        continue
    print(numero)

Pregunta

¿Por qué el 3 no se imprime pero el bucle sí sigue con el resto de números?

Reto 8 · Input y validación

Código

texto = ""

while texto != "python":
    texto = input("Escribe python: ")

print("Correcto")

Prueba obligatoria

Ejecuta el programa con una palabra incorrecta y luego con python. Observa cuándo sigue y cuándo termina.

Método rápido para comprobar un bucle

Qué hacer antes de dar un ejercicio por terminado.

Paso a paso

  1. identifica qué bloque se repite
  2. localiza qué variable cambia
  3. simula las primeras iteraciones
  4. comprueba cuándo termina
  5. piensa si podría quedarse atrapado

Regla útil

Si no puedes seguir mentalmente dos o tres iteraciones, todavía no has entendido del todo el bucle.
