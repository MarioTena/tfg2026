3.5 · Mini-retos: integrar for, while, break y continue

Objetivo de esta parte

Aquí ya no se trata solo de entender la sintaxis. Se trata de resolver pequeños problemas combinando ideas del tema.

Mini-reto 1 · Contar hacia atrás

Muestra una cuenta atrás del 5 al 1. 

for numero in range(5, 0, -1):
    print(numero)

Mini-reto 2 · Sumar del 1 al 10

suma = 0

for numero in range(1, 11):
    suma = suma + numero

print(suma)

Mini-reto 3 · Mostrar solo impares

for numero in range(1, 11):
    if numero % 2 == 0:
        continue
    print(numero)

Mini-reto 4 · Detener al encontrar un valor

for letra in "programacion":
    if letra == "g":
        break
    print(letra)

Mini-reto 5 · Validación con while

Repite hasta que el usuario escriba la palabra python. 

texto = ""

while texto != "python":
    texto = input("Escribe python: ")

print("Correcto")

Mini-reto 6 · Acumulación con while

contador = 1
suma = 0

while contador <= 5:
    suma = suma + contador
    contador = contador + 1

print(suma)

Cómo pensar estos ejercicios

  1. identifica qué se repite
  2. decide si conviene for o while
  3. piensa qué variable cambia
  4. define cuándo termina
  5. si hace falta, añade break o continue

Pregunta de entrevista

¿Cómo decides si usar for o while?

Respuesta sólida: for cuando recorro una secuencia o una repetición estructurada; while cuando dependo de una condición que puede cambiar y no sé tan claramente cuántas iteraciones habrá.
