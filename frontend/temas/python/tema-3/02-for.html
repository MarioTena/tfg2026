3.2 · Bucle for: recorrer y repetir con precisión

¿Qué hace un for?

El bucle for recorre una secuencia elemento a elemento. En cada iteración, una variable toma el siguiente valor de esa secuencia. 

for numero in range(5):
    print(numero)

Aquí Python toma valores de range(5) y los va asignando a numero.

Sintaxis básica

for variable in secuencia:
    bloque

Las piezas importantes son:

  • for
  • una variable que cambia en cada iteración
  • una secuencia o conjunto de valores
  • un bloque indentado

range(): herramienta fundamental

range() genera una secuencia de números muy útil para los bucles. 

for numero in range(5):
    print(numero)

Esto imprime: 

0
1
2
3
4
range(5) empieza en 0 y llega hasta 4. El límite superior no se incluye.

Variantes de range()

range(fin)

range(5)

Genera: 0, 1, 2, 3, 4

range(inicio, fin)

range(2, 6)

Genera: 2, 3, 4, 5

range(inicio, fin, salto)

range(0, 10, 2)

Genera: 0, 2, 4, 6, 8

Mostrar números en secuencia

for numero in range(1, 6):
    print(numero)

Muy útil para tablas, repeticiones contadas y patrones numéricos.

Recorrer texto

for letra in "python":
    print(letra)

Un for no solo sirve con números. También puede recorrer cadenas carácter a carácter.

Usar el valor del bucle para calcular

for numero in range(1, 6):
    print(numero * 2)

En cada iteración puedes operar con la variable y producir resultados nuevos.

Acumular resultados

Muchos problemas con bucles consisten en acumular información. Por ejemplo, sumar números. 

suma = 0

for numero in range(1, 6):
    suma = suma + numero

print(suma)

Aquí suma va cambiando en cada iteración.

Seguir mentalmente una variable acumuladora

Una de las habilidades más importantes es simular cómo cambia una variable. 

suma = 0

for numero in range(1, 4):
    suma = suma + numero
    print("numero:", numero, "suma:", suma)

Esto te enseña a pensar el estado del programa durante el bucle.

⚠️ Error típico: pensar que el límite final se incluye

for numero in range(1, 5):
    print(numero)

Esto imprime 1, 2, 3 y 4. No imprime 5. Este error aparece continuamente al empezar.

⚠️ Error típico: cambiar mal la intención del rango

A veces alguien quiere imprimir del 1 al 10 y escribe esto: 

for numero in range(10):
    print(numero)

Eso imprime del 0 al 9. Si quieres del 1 al 10, debes ajustar el rango. 

for numero in range(1, 11):
    print(numero)

💭 Piensa antes de ejecutar

for numero in range(2, 8, 2):
    print(numero)
  • ¿qué números aparecen?
  • ¿cuántas iteraciones hay?
  • ¿por qué no aparece el 8?

Buenas prácticas con for

  • usa nombres claros: numero, letra, contador
  • no compliques innecesariamente el bloque interno
  • si acumulas algo, inicializa la variable antes del bucle
  • comprueba siempre inicio, final y salto

Mini-reto guiado

Muestra los números del 10 al 1 en orden descendente. 

for numero in range(10, 0, -1):
    print(numero)

Reto de entrevista

Explica qué hace este código: 

total = 0
for numero in range(1, 4):
    total = total + numero
print(total)

Respuesta sólida: suma 1 + 2 + 3 y muestra 6. La variable total acumula el resultado iteración a iteración.
