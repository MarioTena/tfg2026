Bucle for

Aprende a recorrer secuencias y repetir acciones con precisión usando for y range().

Qué vas a entender hoy

El bucle for sirve para recorrer una secuencia y ejecutar un bloque de código en cada paso.

Objetivo Recorrido

Qué problema resuelve

Muchas veces necesitas recorrer números, letras o valores uno a uno. for te permite hacerlo de forma ordenada y clara.

Números

Mostrar del 1 al 10.

Texto

Recorrer palabra por palabra o letra por letra.

Cálculo

Multiplicar, sumar o transformar cada valor.

Acumulación

Ir construyendo un resultado paso a paso.

Idea clave: for no repite “porque sí”; repite recorriendo una secuencia concreta.

Qué debes vigilar

  • qué secuencia estás recorriendo
  • qué valor toma la variable en cada iteración
  • si el rango empieza donde tú crees
  • si el límite final se incluye o no
Piensa así:
la clave del for no es repetir, sino recorrer con orden.

Qué hace un for

for numero in range(5):
    print(numero)

Python toma valores de range(5) y los va asignando a numero uno a uno.

Sintaxis básica

for variable in secuencia:
    bloque

for

La palabra clave que inicia el bucle.

Variable

Cambia en cada iteración.

Secuencia

Es el conjunto de valores que vas a recorrer.

Bloque

Es el código indentado que se ejecuta en cada vuelta.

Dónde aparece for en problemas reales

Recorrer texto

Analizar letra a letra una palabra o una frase.

Secuencias numéricas

Generar series, tablas o conteos.

Informes

Mostrar varios resultados de forma repetida.

Cálculos

Sumar o transformar varios valores de manera ordenada.

range(): herramienta fundamental

for numero in range(5):
    print(numero)
0
1
2
3
4
range(5) empieza en 0 y llega hasta 4. El límite superior no se incluye.

Variantes de range()

range(fin)

Ejemplo: range(5) → 0, 1, 2, 3, 4

range(inicio, fin)

Ejemplo: range(2, 6) → 2, 3, 4, 5

range(inicio, fin, salto)

Ejemplo: range(0, 10, 2) → 0, 2, 4, 6, 8

Usos típicos de for

Mostrar números

for numero in range(1, 6):
    print(numero)

Recorrer texto

for letra in "python":
    print(letra)

Calcular

for numero in range(1, 6):
    print(numero * 2)

Acumular resultados

Muchos ejercicios con bucles consisten en ir guardando un resultado que cambia en cada iteración. 

suma = 0

for numero in range(1, 6):
    suma = suma + numero

print(suma)

Aquí suma va creciendo vuelta a vuelta.

Seguir mentalmente una variable acumuladora

suma = 0

for numero in range(1, 4):
    suma = suma + numero
    print("numero:", numero, "suma:", suma)

Esto te obliga a pensar cómo cambia el estado del programa en cada iteración.

Cómo pensar un for antes de escribirlo

Paso 1

Decide qué secuencia vas a recorrer.

Paso 2

Piensa qué valor tomará la variable en cada iteración.

Paso 3

Comprueba si el final se incluye o no.

Paso 4

Revisa si necesitas inicio, fin y salto.

Con for, muchos fallos vienen de entender mal el rango, no del bucle en sí.

Piensa antes de ejecutar

Haz una simulación mental del rango antes de probarlo.

Código

for numero in range(2, 8, 2):
    print(numero)

Preguntas

  • ¿qué números aparecen?
  • ¿cuántas iteraciones hay?
  • ¿por qué no aparece el 8?
En range(), el final sigue sin incluirse aunque haya salto.

Mini-reto guiado

Controla un rango en sentido descendente.

Reto Fácil

Tu misión

Muestra los números del 10 al 1 en orden descendente.

Resolver en playground

Pista

Si quieres bajar, el salto debe ser negativo.

Mostrar solución 
for numero in range(10, 0, -1):
    print(numero)

Errores frecuentes

Error 1

Pensar que el límite final sí se incluye.

Error 2

Querer del 1 al 10 y escribir range(10).

Error 3

No entender qué valores produce el salto.

Error 4

Olvidar inicializar una variable acumuladora antes del bucle.

for numero in range(1, 5):
    print(numero)

Esto imprime 1, 2, 3 y 4. No imprime 5.

Buenas prácticas

Nombres claros

Usa numero, letra o contador si describen bien la iteración.

Bloque limpio

No mezcles demasiadas cosas dentro del mismo bucle al principio.

Acumuladores

Inicialízalos antes del bucle, no dentro.

Comprueba el rango

Revisa siempre inicio, fin y salto antes de ejecutar.
