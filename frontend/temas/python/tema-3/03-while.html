3.3 · Bucle while: repetir mientras una condición sea verdadera

¿Qué hace un while?

El bucle while repite un bloque de código mientras una condición sea verdadera. Se usa cuando no piensas tanto en “recorrer una secuencia”, sino en “seguir mientras ocurra algo”. 

contador = 1

while contador <= 5:
    print(contador)
    contador = contador + 1

Sintaxis básica

while condicion:
    bloque

Igual que con if y for, la indentación define el bloque.

La idea crítica del while

En casi todo while debe existir algo que cambie durante el bucle. Si nada cambia y la condición sigue siendo verdadera, el bucle no termina.

El mayor peligro de while es el bucle infinito.

Ejemplo paso a paso

contador = 1

while contador <= 3:
    print("Valor actual:", contador)
    contador = contador + 1

print("Fin")

Iteración 1: contador vale 1. Iteración 2: contador vale 2. Iteración 3: contador vale 3. Después pasa a 4, la condición deja de cumplirse y el bucle termina.

Cuándo usar while

  • cuando no sabes cuántas repeticiones habrá
  • cuando dependes de una condición cambiante
  • cuando validas entradas
  • cuando esperas a que ocurra algo

Ejemplo clásico: pedir hasta que sea válido

respuesta = ""

while respuesta != "si":
    respuesta = input("Escribe si para continuar: ")

print("Continuamos")

Este patrón es muy común: repetir hasta que se cumpla una condición deseada.

⚠️ Bucle infinito

contador = 1

while contador <= 5:
    print(contador)

Este código nunca actualiza contador. Como la condición sigue siendo verdadera, el bucle no termina. 

contador = 1

while contador <= 5:
    print(contador)
    contador = contador + 1

Contadores y acumuladores en while

Igual que en for, puedes usar variables que cuentan o acumulan información. 

contador = 1
suma = 0

while contador <= 4:
    suma = suma + contador
    contador = contador + 1

print(suma)

for vs while

  • for: cuando recorres algo o tienes una repetición más estructurada
  • while: cuando dependes de una condición y el número de iteraciones no está tan claro

Ambos pueden resolver algunos problemas similares, pero no siempre tienen la misma claridad.

💭 Piensa antes de ejecutar

contador = 3

while contador > 0:
    print(contador)
    contador = contador - 1
  • ¿qué imprime?
  • ¿cuántas iteraciones hace?
  • ¿cuál es el valor final de contador?

Errores frecuentes

  • olvidar actualizar la variable de control
  • escribir una condición que nunca deja de cumplirse
  • cambiar la variable equivocada
  • no pensar en el valor final del contador

Buenas prácticas con while

  • deja clara la condición
  • asegúrate de que algo cambia dentro del bucle
  • prueba el caso de salida
  • si el bucle es complejo, añade mensajes de seguimiento mientras aprendes

Mini-reto guiado

Imprime los números pares del 2 al 10 usando while. 

numero = 2

while numero <= 10:
    print(numero)
    numero = numero + 2

Reto de entrevista

¿Por qué este código entra en un bucle infinito? 

x = 1
while x < 5:
    print(x)

Respuesta sólida: porque x nunca cambia; por tanto la condición x < 5 siempre sigue siendo verdadera.
