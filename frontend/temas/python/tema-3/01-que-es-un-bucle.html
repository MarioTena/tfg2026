Qué vas a entender hoy

Un bucle permite repetir un bloque de código varias veces sin escribirlo una y otra vez. Es una de las ideas más importantes de la programación.

Objetivo Automatización

Qué problema resuelve

Sin bucles, tendrías que copiar muchas veces la misma instrucción. Con un bucle, escribes la lógica una sola vez y dejas que Python la repita.

Secuencias

Mostrar números del 1 al 100 sin escribir 100 líneas.

Texto

Recorrer letra a letra una palabra o una frase.

Validación

Repetir una pregunta hasta que el usuario responda bien.

Procesos

Aplicar la misma operación a muchos datos.

Idea clave: un bucle no es “copiar código automáticamente”, sino repetir una lógica siguiendo una regla.

Qué debes vigilar

  • qué bloque se repite
  • qué cambia en cada vuelta
  • cuándo sigue repitiéndose
  • cuándo termina
Si no sabes cuándo empieza y cuándo termina un bucle, todavía no lo has entendido bien.

Qué es un bucle

print(1)
print(2)
print(3)
print(4)
print(5)

Esto funciona, pero no escala bien. Si quisieras mostrar muchos más valores, copiar y pegar sería una mala solución. 

for numero in range(5):
    print(numero)

Con un bucle, Python repite el bloque automáticamente.

Dos grandes tipos de bucles

for

Se usa cuando recorres una secuencia o sabes mejor la estructura de la repetición.

while

Se usa cuando repites mientras una condición siga siendo verdadera.

Ambos repiten, pero no piensan igual: for recorre; while continúa mientras una condición siga viva.

Qué significa iteración

Cada vuelta de un bucle se llama iteración.

for numero in range(3):
    print("Iteración")
    print(numero)

Aquí hay 3 iteraciones. En cada una se ejecuta el bloque interior.

Qué suele cambiar en cada iteración

En muchos bucles hay una variable que va cambiando y guía el proceso.

for letra in "sol":
    print(letra)

En cada iteración, letra toma un valor distinto.
