Bucle while

Aprende a repetir un bloque de código mientras una condición siga siendo verdadera.

Qué vas a entender hoy

El bucle while repite código mientras una condición sea verdadera. Se usa cuando lo importante no es recorrer una secuencia, sino seguir mientras ocurra algo.

Objetivo Control

Qué problema resuelve

A veces no sabes exactamente cuántas repeticiones habrá. Lo que sí sabes es que el proceso debe continuar mientras se cumpla una condición.

Contador

Seguir mientras un valor no supere un límite.

Validación

Repetir hasta que el usuario escriba una respuesta válida.

Espera

Continuar hasta que ocurra un evento o cambie un estado.

Condición

El bucle depende de una pregunta lógica, no de una secuencia prefijada.

Idea clave: while no piensa en “cuántas veces”, sino en “mientras esto siga siendo cierto”.

Qué debes vigilar

  • cuál es la condición del bucle
  • qué variable cambia dentro
  • cómo se alcanza la salida
  • si existe riesgo de bucle infinito
Piensa así:
casi todo while necesita algo que cambie en cada vuelta. Si nada cambia, no terminará.

Qué hace un while

contador = 1

while contador <= 5:
    print(contador)
    contador = contador + 1

El bloque se repite mientras contador <= 5 siga siendo verdadero.

Sintaxis básica

while condicion:
    bloque

Igual que con if y for, la indentación define el bloque.

Dónde aparece while en problemas reales

Validación

Repetir hasta que el usuario introduzca un dato correcto.

Esperas

Seguir hasta que ocurra un evento o cambie una condición.

Contadores

Avanzar mientras no se alcance un límite.

Procesos abiertos

Casos donde no sabes exactamente cuántas iteraciones habrá.

La idea crítica del while

Condición

El bucle sigue mientras la condición sea True.

Cambio

Dentro del bucle suele haber una variable que cambia.

Salida

Ese cambio debe acabar haciendo que la condición sea falsa.

Peligro

Si no ocurre ese cambio, puedes crear un bucle infinito.

Ejemplo paso a paso

contador = 1

while contador <= 3:
    print("Valor actual:", contador)
    contador = contador + 1

print("Fin")

Iteración 1: contador vale 1.
Iteración 2: contador vale 2.
Iteración 3: contador vale 3.
Después pasa a 4, la condición deja de cumplirse y el bucle termina.

Cuándo usar while

Repeticiones no cerradas

Cuando no sabes de antemano cuántas vueltas habrá.

Dependencia de estado

Cuando repites mientras algo siga pasando.

Validación

Cuando pides datos hasta que la respuesta sea correcta.

Esperas

Cuando el proceso continúa hasta que cambie una condición.

Ejemplo clásico: pedir hasta que sea válido

respuesta = ""

while respuesta != "si":
    respuesta = input("Escribe si para continuar: ")

print("Continuamos")

Este patrón es muy común: repetir hasta que se cumpla una condición deseada.

Contadores y acumuladores en while

contador = 1
suma = 0

while contador <= 4:
    suma = suma + contador
    contador = contador + 1

print(suma)

Igual que en for, aquí puedes contar, sumar o construir resultados paso a paso.

for vs while

for

Mejor cuando recorres algo o tienes una repetición más estructurada.

while

Mejor cuando dependes de una condición cambiante.

Idea práctica

A veces ambos sirven, pero uno puede ser más claro que el otro.

Cómo pensar un while antes de escribirlo

Paso 1

Define la condición que mantiene vivo el bucle.

Paso 2

Identifica qué variable o estado cambia dentro.

Paso 3

Comprueba cómo se alcanza la salida.

Paso 4

Pregúntate si puede quedarse infinito por error.

Si no puedes explicar por qué el while termina, no deberías ejecutarlo todavía.

Piensa antes de ejecutar

Visualiza el contador y la condición en cada vuelta.

Código

contador = 3

while contador > 0:
    print(contador)
    contador = contador - 1

Preguntas

  • ¿qué imprime?
  • ¿cuántas iteraciones hace?
  • ¿cuál es el valor final de contador?
En while, seguir la variable de control es obligatorio para no perderte.

Reto Fácil

Tu misión

Imprime los números pares del 2 al 10 usando while.

Resolver en playground

Pista

Empieza por 2 y ve sumando 2 en cada iteración.

Mostrar solución 
numero = 2

while numero <= 10:
    print(numero)
    numero = numero + 2

Errores frecuentes

Error 1

Olvidar actualizar la variable de control.

Error 2

Escribir una condición que nunca deja de cumplirse.

Error 3

Cambiar la variable equivocada dentro del bucle.

Error 4

No pensar en el valor final del contador.

contador = 1

while contador <= 5:
    print(contador)

Este código entra en bucle infinito porque contador nunca cambia.

Buenas prácticas

Condición clara

La salida del bucle debe ser comprensible a simple vista.

Cambio real

Asegúrate de que algo cambie dentro del bucle.

Prueba la salida

Comprueba el momento en el que la condición deja de cumplirse.

Depuración

Mientras aprendes, imprime valores intermedios si el comportamiento no te cuadra.
