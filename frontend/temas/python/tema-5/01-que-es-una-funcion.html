5.1 · Qué es una función: reutilizar lógica y escribir código profesional

¿Qué problema resuelven las funciones?

A medida que los programas crecen, empiezan a aparecer bloques de código que se repiten. Copiar y pegar puede parecer rápido al principio, pero empeora el código:

  • duplica lógica
  • dificulta el mantenimiento
  • aumenta errores
  • hace el programa más difícil de leer
Una función permite encapsular una tarea en un bloque reutilizable con nombre propio.

¿Qué es una función?

Una función es un bloque de código que realiza una tarea concreta. Se define una vez y puede ejecutarse tantas veces como sea necesario. 

def saludar():
    print("Hola")

Aquí hemos definido una función llamada saludar. Pero definirla no significa ejecutarla.

Definir no es lo mismo que llamar

def saludar():
    print("Hola")

saludar()

La primera parte crea la función. La segunda la llama para que se ejecute.

Sintaxis básica

def nombre_funcion():
    bloque

Elementos importantes:

  • def indica que vas a definir una función
  • el nombre describe la tarea
  • los paréntesis son obligatorios
  • el bloque va indentado

Por qué las funciones son tan importantes

  • permiten reutilizar código
  • dividen problemas grandes en partes pequeñas
  • mejoran la legibilidad
  • facilitan probar y depurar
  • son una base esencial para escribir software real

Ejemplo de problema sin función

print("Hola Ana")
print("Bienvenida al sistema")

print("Hola Luis")
print("Bienvenido al sistema")

Aquí hay repetición. La estructura es la misma.

Idea de función como herramienta reutilizable

Piensa en una función como una pequeña máquina: le das un nombre, sabes qué hace, y la usas cuando la necesitas.

Funciones incorporadas y funciones propias

Ya has usado funciones aunque quizá no lo notaras: 

print("Hola")
len([1, 2, 3])

print() y len() son funciones incorporadas de Python. Ahora empiezas a crear las tuyas.

💭 Piensa antes de ejecutar

def mostrar_mensaje():
    print("Estoy dentro de la función")

print("Antes")
mostrar_mensaje()
print("Después")
  • ¿qué se imprime primero?
  • ¿qué línea provoca la ejecución de la función?

Errores frecuentes

  • olvidar los paréntesis al llamar a la función
  • pensar que definir una función la ejecuta automáticamente
  • poner mal la indentación
  • elegir nombres poco claros

Buenas prácticas

  • usa nombres que indiquen acción: saludar, calcular_total, mostrar_menu
  • haz que cada función tenga una responsabilidad clara
  • evita funciones demasiado largas al empezar

Mini-reto guiado

Crea una función llamada mostrar_bienvenida que imprima dos líneas de saludo. 

def mostrar_bienvenida():
    print("Bienvenido")
    print("Vamos a aprender funciones")

mostrar_bienvenida()

Pregunta de entrevista

¿Qué es una función y para qué sirve?

Respuesta sólida: una función es un bloque de código reutilizable que encapsula una tarea concreta y ayuda a organizar mejor el programa.
