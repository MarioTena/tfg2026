Cómo trabajar esta pantalla

Clave del tema

Antes de cerrar este bloque

  • has definido funciones sin errores de sintaxis
  • distingues entre parámetros, argumentos y valor de retorno
  • sabes cuándo usar print y cuándo usar return
  • entiendes mejor el ámbito de las variables
  • has probado varias funciones con datos distintos

Última recomendación

No memorices funciones “de receta”. Lo importante es entender qué tarea tienen, qué reciben y qué deberían devolver.
