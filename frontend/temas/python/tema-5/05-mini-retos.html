Qué entrenas aquí

Aquí vas a combinar lo aprendido para empezar a diseñar funciones útiles, no solo ejemplos aislados.

Funciones Parámetros return

Objetivo

El objetivo es que entiendas qué tarea cumple una función, qué recibe y qué devuelve.

Idea clave: una buena función hace una tarea concreta y se entiende con solo leer su nombre.

Cómo usar esta pantalla

  • lee el reto
  • identifica qué debe hacer la función
  • decide si debe imprimir o devolver
  • pruébalo en el playground y cambia entradas
No confundas mostrar un resultado con devolverlo. Esa diferencia cambia totalmente cómo reutilizas una función.

Misiones rápidas

Mini-reto 1 · Función que saluda

Crea una función simple que use un parámetro.

def saludar(nombre):
    print("Hola", nombre)

saludar("Ana")

Mini-reto 2 · Función que devuelve el doble

Devuelve un valor para poder reutilizarlo.

def doble(numero):
    return numero * 2

print(doble(8))

Mini-reto 3 · Función que clasifica edad

Combina condicionales dentro de una función.

def clasificar_edad(edad):
    if edad < 12:
        return "niño"
    if edad < 18:
        return "adolescente"
    return "adulto"

print(clasificar_edad(15))

Mini-reto 4 · Función que calcula IVA

Encapsula una operación en una función clara.

def calcular_total(precio):
    return precio * 1.21

print(calcular_total(50))

Mini-reto 5 · Función booleana

Devuelve True o False según una regla.

def es_par(numero):
    return numero % 2 == 0

print(es_par(8))
print(es_par(5))

Mini-reto 6 · Reutilizar funciones

Llama varias veces a la misma función con distintos datos.

def cuadrado(numero):
    return numero * numero

resultado1 = cuadrado(3)
resultado2 = cuadrado(5)
print(resultado1)
print(resultado2)

Cómo pensar este tipo de ejercicios

Antes de escribir una función, define bien su contrato.

Paso a paso

  1. define claramente la tarea de la función
  2. decide qué datos necesita recibir
  3. decide si debe imprimir o devolver
  4. elige un nombre claro
  5. prueba varios casos

Regla útil

Si quieres reutilizar el resultado después, normalmente necesitas return, no print.

Pregunta típica de entrevista

Pregunta: ¿Cuándo es preferible usar return en lugar de print?

Respuesta sólida: cuando quieres reutilizar el resultado de la función en otras operaciones, almacenarlo o tomar decisiones con él.
