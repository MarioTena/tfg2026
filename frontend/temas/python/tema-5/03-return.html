return

Aprende a devolver resultados desde una función en lugar de limitarte a imprimirlos.

Nivel inicial · 12-15 min · Resultados reutilizables
return Concepto clave
Empezar misión Abrir playground

Qué vas a entender hoy

Imprimir y devolver no son lo mismo. return hace que una función produzca un resultado reutilizable.

Objetivo Resultados

Qué problema resuelve

Si una función solo imprime, muestra algo por pantalla, pero no te deja trabajar después con ese valor. Con return, puedes guardar el resultado, reutilizarlo y combinarlo.

Cálculos

Guardar el resultado de una suma o de un IVA.

Reutilización

Usar el resultado dentro de otras operaciones.

Decisiones

Devolver True o False desde una función.

Composición

Encadenar funciones y trabajar con sus resultados.

Idea clave: return convierte a la función en una herramienta que produce un valor utilizable.

Qué debes vigilar

  • imprimir no es devolver
  • return termina la función
  • el valor devuelto puede guardarse en una variable
  • si no devuelves nada, el resultado será None
Piensa así:
print() enseña algo; return entrega algo.

Dónde aparece return en problemas reales

Cálculos

Devolver importes, descuentos, medias o resultados numéricos.

Mensajes

Construir texto y usarlo más adelante en otra parte del programa.

Validaciones

Devolver booleanos como True o False.

Funciones encadenadas

Usar el resultado de una función dentro de otra operación.

Imprimir no es lo mismo que devolver

Este es uno de los saltos conceptuales más importantes del curso. Una función puede mostrar algo por pantalla con print(), pero también puede devolver un valor usando return.

Función que imprime

def sumar(a, b):
    print(a + b)

sumar(2, 3)

Esto muestra 5, pero no lo deja disponible para seguir trabajando con él.

Función que devuelve

def sumar(a, b):
    return a + b

resultado = sumar(2, 3)
print(resultado)

Ahora el valor devuelto puede guardarse, reutilizarse o combinarse con otras operaciones.

Qué hace exactamente return

Termina la función

En cuanto se ejecuta, la función deja de seguir leyendo líneas.

Envía un valor

Devuelve un resultado al lugar donde se llamó a la función.

Permite reutilizar

Ese valor puede guardarse, imprimirse o usarse en otra operación.

Mejora diseño

Hace que la función sea más útil y más flexible.

Usar el valor devuelto

def cuadrado(numero):
    return numero * numero

print(cuadrado(4))
resultado = cuadrado(6)
print(resultado + 10)

Error típico: creer que print sustituye a return

def sumar(a, b):
    print(a + b)

resultado = sumar(2, 3)
print(resultado)

Esto imprimirá primero 5 y luego None, porque la función no ha devuelto nada.

Funciones que devuelven texto

def crear_saludo(nombre):
    return "Hola " + nombre

mensaje = crear_saludo("Ana")
print(mensaje)

Funciones que deciden y devuelven

def es_mayor_de_edad(edad):
    return edad >= 18

print(es_mayor_de_edad(20))
print(es_mayor_de_edad(15))

Una función también puede devolver booleanos.

Devolver pronto

def clasificar(numero):
    if numero > 0:
        return "positivo"
    if numero < 0:
        return "negativo"
    return "cero"

En cuanto se ejecuta un return, la función termina.

Cómo pensar un return antes de escribirlo

Paso 1

Pregúntate si la función debe mostrar algo o producir un valor.

Paso 2

Si el resultado debe usarse fuera, probablemente necesitas return.

Paso 3

Decide qué valor concreto va a devolver la función.

Paso 4

Piensa qué parte del programa usará ese resultado después.

Si el resultado de una función debe seguir vivo fuera de ella, necesitas devolverlo.

Piensa antes de ejecutar

Antes de probar, sigue el valor que sale de la función.

Código

def doble(numero):
    return numero * 2

valor = doble(5)
print(valor)
print(doble(3) + 1)

Preguntas

  • ¿qué vale valor?
  • ¿qué imprime la última línea?
Pista: la función devuelve un valor, no lo imprime sola.

Mini-reto guiado

Ahora toca crear una función que produzca un resultado reutilizable.

Reto Muy fácil

Tu misión

Crea una función que reciba un precio y devuelva el precio con IVA del 21%.

Resolver en playground

Pista

Aquí no quieres solo mostrar el resultado. Quieres poder usarlo después.

Mostrar solución 
def precio_con_iva(precio):
    return precio * 1.21

final = precio_con_iva(100)
print(final)

Errores frecuentes

Error 1

Confundir imprimir con devolver.

Error 2

Olvidar guardar o usar el resultado devuelto.

Error 3

Escribir código después de un return pensando que seguirá ejecutándose.

Error 4

No entender por qué aparece None cuando la función no devuelve nada.

Buenas prácticas

Separa mostrar y calcular

Una función puede calcular y otra parte del programa puede imprimir.

Devuelve valores útiles

No uses return “porque sí”; úsalo cuando el resultado deba reutilizarse.

Piensa en el flujo

Recuerda que la función termina justo en el return.

Comprueba el resultado

Haz pruebas guardando el valor devuelto en variables para entenderlo mejor.
