1.2 · Tu primer programa y cómo se ejecuta

Un programa es una secuencia

Python ejecuta el código de arriba hacia abajo, sin saltarse líneas. 

print("Uno")
print("Dos")
print("Tres")
El orden importa. Siempre.

Tu primer programa real

El comando print sirve para mostrar información. 

print("Hola mundo")

Esta línea ya es un programa completo.

💡 Piensa antes de ejecutar

Antes de darle a “Ejecutar”, pregúntate:

  • ¿Qué va a mostrar?
  • ¿Cuántas líneas?
  • ¿En qué orden?

Mini-reto guiado

Escribe un programa que muestre:

  1. Tu nombre
  2. Tu edad
  3. Un mensaje final
# Ejemplo de estructura
print("Nombre")
print("Edad")
print("Mensaje final")
No busques hacerlo perfecto. El objetivo es probar y equivocarte.

Errores = aprendizaje

Prueba a cometer un error a propósito: 

print(Hola)

Lee el error que aparece. Aprender a interpretar errores es parte fundamental de programar.
