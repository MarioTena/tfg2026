1.7 · Mini proyecto: tu primer programa completo

🎯 Objetivo

Ahora vas a crear tu primer programa “real”. No será solo una línea suelta. Será un pequeño programa con sentido.

Aquí mezclas TODO lo aprendido: variables, tipos, operaciones y mensajes.

📘 Escenario

Vas a crear un pequeño programa que calcule la edad aproximada que tendrá una persona dentro de 10 años.

El programa debe:

  • Guardar un nombre en una variable
  • Guardar la edad actual en otra variable
  • Calcular la edad dentro de 10 años
  • Mostrar un mensaje completo por pantalla

🧩 Estructura base

# Paso 1: variables
nombre = "Tu nombre"
edad_actual = 0

# Paso 2: cálculo
edad_futura = edad_actual + 10

# Paso 3: mensaje final
print("Hola,", nombre)
print("Dentro de 10 años tendrás", edad_futura, "años")
Cambia valores. Prueba edades distintas. Observa cómo cambia el resultado.

🔥 Nivel 2 (opcional)

Añade una variable nueva: 

año_actual = 2024
año_futuro = año_actual + 10

Y muestra también el año: 

print("En el año", año_futuro, "tendrás", edad_futura, "años")
Ya estás combinando datos numéricos y texto.

🚀 Nivel 3 (reto creativo)

Modifica el programa para que:

  • Use al menos 4 variables
  • Haga al menos 2 operaciones matemáticas
  • Muestre 3 mensajes diferentes
No copies. Piensa. Diseña tu propio pequeño programa.

🧠 Reflexión final

Si puedes hacer este mini proyecto sin copiar, significa que:

  • Entiendes qué es una variable
  • Sabes hacer operaciones
  • Sabes mostrar resultados
  • Has aprendido a leer errores
Esto ya es programar. Lo demás es ampliar complejidad.
