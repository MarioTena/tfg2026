Un string parece simple, pero muchos errores iniciales vienen de no pensar bien qué es texto y qué no.

¿Qué es un string?

Un string es texto. En Python, cualquier cosa entre comillas se interpreta como texto. 

nombre = "Ana"
mensaje = "Hola mundo"
Si lleva comillas, Python lo trata como texto.

Comillas simples o dobles

texto1 = "Hola"
texto2 = 'Hola'

Ambas formas funcionan. Lo importante es abrir y cerrar correctamente.

Comillas dobles

"Hola"

Comillas simples

'Hola'

Regla real

No importa tanto cuál uses, sino cerrarlas bien.

Error típico

Abrir con una y cerrar mal provoca error de sintaxis.

Piensa antes de ejecutar

print("Hola" + "Mundo")
  • ¿Qué se mostrará?
  • ¿Aparecerá un espacio entre las palabras?
Python no añade espacios automáticamente cuando unes strings con +.

Concatenar texto

El operador + une strings. 

print("Hola" + " " + "Mundo")

También puedes usar comas con print(): 

print("Hola", "Mundo")

Con +

Tú controlas cada espacio y cada parte del texto.

Con comas

Python separa automáticamente los elementos con espacios.

Usar variables con texto

nombre = "Lucía"
print("Hola", nombre)

Aquí Python toma el valor de nombre y lo inserta en la salida.

No estás mostrando la palabra nombre, sino el texto guardado dentro.

Error típico: mezclar texto y número

edad = 20
print("Tengo " + edad + " años")

Este código falla porque intenta unir directamente un string y un entero.

Forma correcta:

print("Tengo", edad, "años")
Cuando mezclas tipos distintos, no basta con “querer unirlos”. Python necesita que pienses qué tipo de dato estás usando.

Longitud de un string

nombre = "Carlos"
print(len(nombre))

La función len() devuelve cuántos caracteres tiene un texto.

También cuenta espacios.

Acceder a letras individuales

nombre = "Ana"
print(nombre[0])
print(nombre[1])

Python empieza a contar desde 0.

0

Primera letra

1

Segunda letra

2

Tercera letra

Regla

Los índices empiezan en 0, no en 1.
