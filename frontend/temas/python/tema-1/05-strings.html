1.5 · Strings: trabajar con texto

¿Qué es un string?

Un string es texto. Cualquier cosa que esté entre comillas. 

nombre = "Ana"
mensaje = "Hola mundo"
Si lleva comillas → es texto.

Comillas simples o dobles

texto1 = "Hola"
texto2 = 'Hola'

Ambas funcionan igual.

Lo importante es abrir y cerrar correctamente.

💭 Piensa antes de ejecutar

print("Hola" + "Mundo")
  • ¿Qué mostrará?
  • ¿Habrá espacio entre palabras?
Python no añade espacios automáticamente.

Concatenar texto

print("Hola" + " " + "Mundo")

El símbolo + une strings.

Pero hay una forma más sencilla: 

print("Hola", "Mundo")
Con comas, Python añade espacios automáticamente.

Usar variables con texto

nombre = "Lucía"
print("Hola", nombre)

Python sustituye la variable por su valor.

⚠️ Error típico: mezclar texto y número

edad = 20
print("Tengo " + edad + " años")

❓ ¿Funcionará?

No. Python no puede sumar texto y número directamente.

Forma correcta:

print("Tengo", edad, "años")

Longitud de un string

nombre = "Carlos"
print(len(nombre))

La función len() devuelve cuántos caracteres tiene el texto.

También cuenta espacios.

Acceder a letras individuales

nombre = "Ana"
print(nombre[0])
print(nombre[1])

Python empieza a contar desde 0.

0 → primera letra 1 → segunda 2 → tercera

Mini-reto guiado

Crea un programa que:

  1. Guarde tu nombre
  2. Muestre cuántas letras tiene
  3. Muestre la primera letra
# Estructura base
nombre = "Tu nombre"

print("Tu nombre tiene", len(nombre), "letras")
print("La primera letra es:", nombre[0])
Cambia el nombre y observa cómo cambia el resultado.
