1.1 · ¿Qué es Python y por qué se usa tanto?

Python es una herramienta para dar órdenes

Python es un lenguaje de programación, es decir, una forma de escribir instrucciones para que un ordenador las ejecute.

El ordenador no interpreta, no adivina y no corrige: ejecuta exactamente lo que le dices.

¿Por qué aprender Python?

🧠 Fácil de leer

Su sintaxis se parece al lenguaje humano.

🚀 Muy usado

Web, IA, datos, automatización, educación.

⚡ Ideal para aprender

Permite centrarse en la lógica, no en la complejidad técnica.

Python es interpretado

Python ejecuta el código línea a línea, en orden. No hay compilación previa como en otros lenguajes. 

print("Empiezo")
print("Sigo")
print("Termino")

Si hay un error, Python se detiene en ese punto.

Piensa como programador

Aprender Python no es memorizar comandos. Es aprender a:

  • Dar instrucciones claras
  • Pensar paso a paso
  • Detectar errores
Programar es aprender a pensar con precisión.
