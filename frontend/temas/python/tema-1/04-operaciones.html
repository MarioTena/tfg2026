1.4 · Operaciones: cómo calcula Python

Python también calcula

Además de guardar información, Python puede realizar operaciones matemáticas. 

print(5 + 3)
Python evalúa la operación y muestra el resultado.

Operaciones básicas

a = 10
b = 3

print(a + b)   # suma
print(a - b)   # resta
print(a * b)   # multiplicación
print(a / b)   # división
print(a // b)  # división entera
print(a % b)   # resto
print(a ** b)  # potencia

No memorices aún. Lo importante es entender qué hace cada símbolo.

💭 Piensa antes de ejecutar

print(10 / 3)
  • ¿Qué resultado esperas?
  • ¿Será 3 o 3.333...?
Python siempre devuelve decimal en divisiones normales.

División normal vs división entera

print(10 / 3)
print(10 // 3)

Observa la diferencia:

  • / → devuelve decimal
  • // → elimina los decimales
Esta diferencia es clave en muchos algoritmos.

Orden de operaciones

print(2 + 3 * 4)
  • ¿Da 20?
  • ¿O da 14?
Python respeta las reglas matemáticas: primero multiplicación, luego suma. 
print((2 + 3) * 4)

⚠️ Error típico: mezclar tipos

edad = 20
print("Tengo " + edad + " años")

❓ ¿Funcionará?

No. No se puede sumar texto con número directamente.

Forma correcta:

print("Tengo", edad, "años")

Mini-reto práctico

Crea un programa que:

  1. Guarde dos números en variables
  2. Calcule su suma
  3. Calcule su multiplicación
  4. Muestre ambos resultados
# Estructura base
a = 0
b = 0

print("Suma:", a + b)
print("Multiplicación:", a * b)
Cambia los números y observa cómo varían los resultados.
