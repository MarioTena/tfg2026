Idea clave: una variable no es “una palabra especial”. Es un nombre que apunta a un valor.

Qué debes vigilar

  • qué nombre tiene la variable
  • qué valor guarda
  • si el valor cambia más adelante
  • qué tipo de dato estás usando
Si confundes el nombre de la variable con el valor que contiene, te costará mucho entender el código.

¿Qué es una variable?

Una variable es una forma de guardar información con un nombre para poder reutilizarla después. 

edad = 20

Aquí, edad es el nombre de la variable y 20 es el valor guardado.

Piensa en una variable como una etiqueta con un valor asociado.

Usar una variable

Una vez has guardado un valor, puedes utilizarlo en otras líneas del programa. 

edad = 20
print(edad)

Python no imprime el texto edad. Imprime el valor que esa variable contiene.

Piensa antes de ejecutar

nombre = "Ana"
print(nombre)
  • ¿Qué se mostrará?
  • ¿Se imprime la palabra nombre o el valor guardado?
Antes de ejecutar, intenta responder mentalmente. Después compruébalo.

Usar variables en mensajes

Las variables sirven para construir programas algo más flexibles. 

nombre = "Ana"
print("Me llamo", nombre)

Python sustituye la variable por su contenido cuando ejecuta el programa.

Tipos de datos básicos

No todos los valores son iguales. Python distingue entre distintos tipos de datos. 

edad = 21        # entero
altura = 1.70    # decimal
nombre = "Luis"  # texto
activo = True    # booleano

Enteros

Números sin decimales, como 3 o 21.

Decimales

Números con parte decimal, como 1.5 o 2.75.

Strings

Texto entre comillas, como "Hola".

Booleanos

Valores lógicos: True o False.

No hace falta declarar el tipo manualmente. Python lo deduce por ti.

Una variable puede cambiar

Las variables no tienen por qué quedarse siempre con el mismo valor. 

edad = 18
edad = 19
print(edad)

El valor final será 19, porque la segunda asignación sustituye a la primera.

Error típico: usar una variable antes de crearla

print(nombre)
nombre = "Ana"

Este código falla porque Python ejecuta de arriba hacia abajo y en la primera línea todavía no existe ninguna variable llamada nombre.

Una variable solo puede usarse después de haber sido creada.
