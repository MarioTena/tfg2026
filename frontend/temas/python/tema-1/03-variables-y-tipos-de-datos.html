1.3 · Variables: guardar información y reutilizarla

¿Qué es una variable?

Una variable es una forma de guardar información para usarla más adelante.

Piensa en una variable como una etiqueta con un valor dentro.

Crear una variable

En Python, crear una variable es muy sencillo: 

edad = 20

Aquí estamos diciendo:

  • El nombre de la variable es edad
  • Su valor es 20

💭 Piensa antes de ejecutar

Antes de probar este código, piensa: 

edad = 20
print(edad)
  • ¿Qué va a mostrar?
  • ¿Mostrará el texto edad o el número?
Ejecuta el código solo cuando tengas una respuesta en mente.

Usar variables en mensajes

Las variables sirven para combinar información: 

nombre = "Ana"
print("Me llamo", nombre)

Python sustituye la variable por su valor.

Tipos de datos (sin memorizar)

Cada valor tiene un tipo. Python lo detecta automáticamente. 

edad = 21        # número entero
altura = 1.70    # número decimal
nombre = "Luis"  # texto
activo = True    # booleano
No tienes que indicar el tipo: Python lo deduce por ti.

⚠️ Error típico (muy importante)

Observa este código: 

print(nombre)
nombre = "Ana"

❓ Pregunta clave:

  • ¿Funcionará?
  • ¿Por qué sí o por qué no?
Python ejecuta de arriba hacia abajo. No puede usar una variable que aún no existe.

Mini-reto guiado

Crea un programa que:

  1. Guarde tu nombre en una variable
  2. Guarde tu edad en otra variable
  3. Muestre un mensaje usando ambas
# Ejemplo de estructura
nombre = "Tu nombre"
edad = 0
print("Me llamo", nombre)
print("Tengo", edad, "años")
