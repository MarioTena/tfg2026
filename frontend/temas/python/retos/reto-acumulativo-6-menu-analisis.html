Qué entrenas aquí

Este reto sirve como puente entre retos y proyectos: te obliga a organizar un programa pequeño con menú y varias acciones.

Menú Funciones Organización

Objetivo

Vas a crear un programa con menú donde el usuario pueda introducir datos, ver un resumen y salir.

Qué conceptos estás combinando

  • while
  • if / elif / else
  • funciones
  • listas
  • validación de opción
  • menús

Enunciado

Crea un programa con menú que permita:

  • añadir un dato a una lista
  • mostrar los datos guardados
  • mostrar un resumen simple
  • salir

El programa debe seguir funcionando hasta que el usuario elija salir.

Requisitos mínimos

  • el menú debe repetirse hasta que el usuario elija salir
  • debes validar las opciones introducidas
  • debes guardar los datos en una lista
  • debes mostrar los datos actuales cuando se solicite
  • debes incluir al menos un resumen simple sobre los datos guardados

Diseño recomendado

Paso 1

Crea una lista vacía y un bucle principal para mantener el menú activo.

Paso 2

Implementa primero las opciones mínimas: añadir dato y salir.

Paso 3

Después añade mostrar datos y el resumen final.

Paso 4

Cuando funcione, separa cada acción en una función distinta para organizar mejor el programa.

Pistas opcionales

Pista 1

Usa un while True para mantener el menú activo.

Pista 2

Crea una función para mostrar el menú.

Pista 3

Valida las opciones con if / elif / else.

Pista 4

Haz primero una versión mínima con solo dos opciones.

Versión ampliada

  • permitir borrar un dato
  • mostrar el número total de elementos
  • evitar entradas vacías
  • separar cada acción en una función distinta
