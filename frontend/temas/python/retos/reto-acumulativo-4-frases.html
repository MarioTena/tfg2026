Qué entrenas aquí

Este reto te ayuda a tratar una frase como una fuente de datos que puedes descomponer y analizar.

Texto Descomposición Funciones

Objetivo

Vas a pedir una frase al usuario y mostrar varias estadísticas sobre ella.

Qué conceptos estás combinando

  • input
  • strings
  • listas de palabras
  • funciones
  • conteos
  • búsqueda de vocales

Enunciado

Crea un programa que pida una frase al usuario y muestre:

  • cuántas palabras tiene
  • cuántas letras tiene sin contar espacios
  • cuántas vocales contiene
  • cuál es la palabra más larga

Pistas opcionales

Pista 1

Puedes usar split() para separar la frase en palabras.

Pista 2

Para contar letras sin espacios, puedes recorrer la frase y saltarte el espacio.

Pista 3

Las vocales pueden compararse con "aeiouáéíóú".

Pista 4

Para la palabra más larga, compara longitudes con len().

Versión ampliada

  • contar también consonantes
  • mostrar cuántas palabras empiezan por vocal
  • ignorar signos de puntuación simples
