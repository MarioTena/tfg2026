Qué entrenas aquí

Este reto te ayuda a tratar una frase como una fuente de datos que puedes descomponer y analizar.

Texto Descomposición Funciones

Objetivo

Vas a pedir una frase al usuario y mostrar varias estadísticas sobre ella.

Qué conceptos estás combinando

  • input
  • strings
  • listas de palabras
  • funciones
  • conteos
  • búsqueda de vocales

Enunciado

Crea un programa que pida una frase al usuario y muestre:

  • cuántas palabras tiene
  • cuántas letras tiene sin contar espacios
  • cuántas vocales contiene
  • cuál es la palabra más larga

Requisitos mínimos

  • debes pedir una frase al usuario
  • debes separar palabras para analizarlas correctamente
  • debes contar letras sin incluir espacios
  • debes contar vocales recorriendo la frase o las palabras
  • debes identificar correctamente la palabra más larga

Diseño recomendado

Paso 1

Pide una frase al usuario y guárdala en una variable.

Paso 2

Sepárala en palabras para contar cuántas hay y localizar la más larga.

Paso 3

Recorre la frase para contar letras reales y vocales.

Paso 4

Muestra el resumen final de forma clara y ordenada.

Fases del reto

Piensa primero en cómo separar el problema: no todo se resuelve con un único recorrido.

Fase 1

Pide la frase al usuario y guárdala sin modificar todavía.

Fase 2

Separa la frase en palabras para contar cuántas hay.

Fase 3

Recorre la frase para contar letras reales sin incluir espacios.

Fase 4

Cuenta cuántas vocales aparecen.

Fase 5

Busca la palabra más larga comparando longitudes.

Fase 6

Mejora mensajes y comprueba cómo responde el programa con frases cortas, largas o con varios espacios.

Pistas opcionales

Pista 1

Puedes usar split() para separar la frase en palabras.

Pista 2

Para contar letras sin espacios, puedes recorrer la frase y saltarte el espacio.

Pista 3

Las vocales pueden compararse con "aeiouáéíóú".

Pista 4

Para la palabra más larga, compara longitudes con len().

Errores frecuentes

Error 1

Contar letras incluyendo espacios cuando el enunciado pide excluirlos.

Error 2

Buscar la palabra más larga sin separar antes la frase en palabras.

Error 3

No normalizar el texto al contar vocales y perder mayúsculas o vocales acentuadas.

Error 4

Intentar resolver todo con una sola variable o un solo recorrido sin distinguir objetivos.

Versión ampliada

  • contar también consonantes
  • mostrar cuántas palabras empiezan por vocal
  • ignorar signos de puntuación simples

Qué deberías haber consolidado al terminar

Este reto sirve para pasar de manipular strings pequeños a analizar texto con varios objetivos a la vez.

Competencias que consolidan

  • separar una frase en palabras
  • distinguir entre recorrer caracteres y recorrer palabras
  • contar letras y vocales con criterio
  • comparar longitudes para elegir un máximo
  • organizar un análisis de texto en pasos claros

Cierre didáctico

Aquí trabajas con texto como estructura de datos: una misma frase puede analizarse desde varias perspectivas si sabes dividir bien el problema.
