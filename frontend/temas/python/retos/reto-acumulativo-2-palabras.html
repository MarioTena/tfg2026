Qué entrenas aquí

Este reto te obliga a tratar palabras como datos reales: guardarlas, compararlas y analizarlas con varias reglas.

Integración Texto Funciones

Objetivo

Vas a pedir palabras al usuario hasta que escriba salir. Después, tu programa deberá analizarlas y mostrar un pequeño resumen.

Idea clave: aquí no trabajas strings de forma aislada. Los conviertes en una colección de datos que luego debes interpretar.

Qué conceptos estás combinando

  • input
  • strings
  • listas
  • funciones
  • while
  • conteo
  • comparación de longitudes
  • repeticiones

Enunciado

Crea un programa que pida palabras al usuario una a una hasta que escriba salir. Guarda todas las palabras en una lista.

Cuando termine la entrada, el programa debe mostrar:

  • cuántas palabras ha introducido
  • cuál es la palabra más larga
  • cuántas palabras empiezan por vocal
  • cuántas palabras están repetidas

Requisitos mínimos

  • debes usar un bucle while
  • debes guardar las palabras en una lista
  • debes usar al menos una función para el análisis
  • debes parar cuando el usuario escriba salir
  • si no se ha introducido ninguna palabra, debes indicarlo claramente
No mezcles demasiadas cosas en un mismo bloque. Primero recoge bien las palabras y luego analiza.

Diseño recomendado

Paso 1

Crea una lista vacía para las palabras.

Paso 2

Pide palabras hasta que aparezca salir.

Paso 3

Guarda cada palabra válida en la lista.

Paso 4

Crea una función para encontrar la palabra más larga.

Paso 5

Crea otra función para contar palabras que empiezan por vocal.

Paso 6

Calcula también cuántas están repetidas.

Pistas opcionales

Pista 1

Usa len() para comparar la longitud de palabras.

Pista 2

Puedes comprobar la primera letra con palabra[0] si la palabra no está vacía.

Pista 3

Para las vocales, piensa en "aeiou".

Pista 4

Para las repetidas, compara cuántas veces aparece cada palabra.

Pista 5

Conviene ignorar mayúsculas y minúsculas al comparar repeticiones.

Pista 6

Antes de analizar, comprueba si la lista está vacía.

Versión ampliada

  • mostrar también la palabra más corta
  • indicar cuántas palabras tienen más de 5 letras
  • mostrar la lista de palabras sin repetir
  • ordenar las palabras por longitud
