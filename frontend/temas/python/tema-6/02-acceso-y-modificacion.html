6.2 · Acceso y modificación: leer y actualizar valores por clave

Acceder a un valor por su clave

Para acceder a un valor en un diccionario, usas la clave entre corchetes. 

persona = {
    "nombre": "Ana",
    "edad": 30,
    "ciudad": "Madrid"
}

print(persona["nombre"])
print(persona["edad"])

Esto devuelve los valores asociados a esas claves.

Modificar un valor existente

persona = {
    "nombre": "Ana",
    "edad": 30
}

persona["edad"] = 31
print(persona)

Si la clave ya existe, su valor se actualiza.

Añadir una nueva clave

persona = {
    "nombre": "Ana",
    "edad": 30
}

persona["email"] = "ana@email.com"
print(persona)

Si la clave no existía, se crea.

⚠️ Error típico: acceder a una clave inexistente

producto = {
    "nombre": "Ratón",
    "precio": 25
}

print(producto["stock"])

Esto da error porque la clave "stock" no existe.

Usar get() para acceder con más seguridad

producto = {
    "nombre": "Ratón",
    "precio": 25
}

print(producto.get("stock"))
print(producto.get("stock", 0))

get() devuelve None o un valor por defecto si la clave no existe. Es muy útil para evitar errores.

Cuando no estás seguro de si una clave existe, get() suele ser mejor que acceder directamente con corchetes.

Comprobar si una clave existe

usuario = {
    "nombre": "Luis",
    "activo": True
}

print("activo" in usuario)
print("email" in usuario)

El operador in permite comprobar si una clave está en el diccionario.

Eliminar una clave

datos = {
    "nombre": "Marta",
    "edad": 28,
    "ciudad": "Sevilla"
}

 del datos["ciudad"]
print(datos)

del elimina la clave y su valor asociado.

alumno = {

alumno["nota"] = 9
alumno["curso"] = "Python"

Errores frecuentes

  • escribir mal una clave
  • acceder a una clave que no existe
  • olvidar que las claves de texto deben ir entre comillas
  • confundir actualizar una clave con añadir una nueva

Mini-reto guiado

Crea un diccionario de producto, cambia el precio y añade una clave stock. 

producto = {
    "nombre": "Monitor",
    "precio": 150
}

producto["precio"] = 175
producto["stock"] = 12
print(producto)

Pregunta de entrevista

¿Qué ventaja tiene get() frente a acceder directamente con diccionario[clave]?

Respuesta sólida: get() permite recuperar un valor sin provocar error si la clave no existe, y además puede devolver un valor por defecto.
