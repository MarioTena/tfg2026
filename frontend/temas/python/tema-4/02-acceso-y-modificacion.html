4.2 · Acceso y modificación: trabajar con posiciones dentro de una lista

Acceder a un elemento

Cada elemento de una lista tiene una posición, llamada índice. En Python, los índices empiezan en 0. 

frutas = ["manzana", "pera", "uva"]
print(frutas[0])
print(frutas[1])
print(frutas[2])

Esto imprimirá manzana, pera y uva.

En Python, el primer elemento no es el 1. Es el 0.

¿Por qué empieza en 0?

Es una convención muy común en programación. Cuesta un poco al principio, pero después se vuelve natural. Debes asumirlo cuanto antes porque aparece constantemente.

Modificar un elemento

Las listas en Python son mutables, lo que significa que sus elementos se pueden cambiar. 

frutas = ["manzana", "pera", "uva"]
frutas[1] = "plátano"
print(frutas)

Ahora la lista será ["manzana", "plátano", "uva"].

Índices negativos

Python también permite acceder desde el final de la lista usando índices negativos. 

numeros = [10, 20, 30, 40]
print(numeros[-1])
print(numeros[-2])
  • -1 es el último elemento
  • -2 es el penúltimo

⚠️ Error típico: índice fuera de rango

frutas = ["manzana", "pera", "uva"]
print(frutas[3])

Este código da error porque esa lista tiene índices 0, 1 y 2. El índice 3 no existe.

Si una lista tiene longitud 3, su último índice positivo es 2.

Usar len() para razonar índices

frutas = ["manzana", "pera", "uva"]
print(len(frutas))

Si la longitud es 3, los índices válidos van de 0 a 2.

Acceder al primer y último elemento

numeros = [5, 8, 12, 20]
print(numeros[0])
print(numeros[-1])

Esta es una operación muy común.

Rebanado básico o slicing

También puedes obtener una parte de la lista. 

numeros = [10, 20, 30, 40, 50]
print(numeros[1:4])

Esto devuelve [20, 30, 40]. Empieza en el índice 1 y llega hasta antes del 4.

💭 Piensa antes de ejecutar

animales = ["gato", "perro", "loro", "pez"]
print(animales[0])
print(animales[-1])
print(animales[1:3])
  • ¿qué imprime cada línea?
  • ¿por qué el último valor no sale en el slicing?

Errores frecuentes

  • pensar que el primer índice es 1
  • usar índices fuera del rango
  • confundir modificar un elemento con añadir uno nuevo
  • no entender que el slicing no incluye el límite final

Mini-reto guiado

Crea una lista con 4 asignaturas, muestra la primera y la última, y cambia la segunda por otra nueva. 

asignaturas = ["Matemáticas", "Historia", "Física", "Lengua"]
print(asignaturas[0])
print(asignaturas[-1])
asignaturas[1] = "Programación"
print(asignaturas)

Pregunta de entrevista

¿Qué significa que una lista sea mutable?

Respuesta sólida: significa que sus elementos se pueden modificar después de crear la lista.
