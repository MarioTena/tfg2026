Cuando una lista ya existe, el siguiente paso natural es saber entrar en ella y modificar partes concretas.

Acceder a un elemento

Cada elemento de una lista tiene una posición, llamada índice. En Python, los índices empiezan en 0. 

frutas = ["manzana", "pera", "uva"]
print(frutas[0])
print(frutas[1])
print(frutas[2])

Esto imprimirá manzana, pera y uva.

En Python, el primer elemento no es el 1. Es el 0.

La lógica de los índices

Al principio cuesta, pero aquí está una de las bases que más vas a usar.

¿Por qué empieza en 0?

Es una convención muy común en programación. Cuesta un poco al principio, pero después se vuelve natural. Debes asumirlo cuanto antes porque aparece constantemente.

Qué debes interiorizar

  • si una lista tiene 3 elementos, sus índices son 0, 1 y 2
  • la longitud no coincide con el último índice
  • el índice representa posición, no cantidad total

Trabajar con posiciones dentro de una lista

Una vez sabes acceder por índice, puedes modificar valores, mirar desde el final y recuperar elementos clave.

Modificar Índices negativos Uso frecuente

1. Modificar un elemento

Las listas en Python son mutables, lo que significa que sus elementos se pueden cambiar después de crear la lista. 

frutas = ["manzana", "pera", "uva"]
frutas[1] = "plátano"
print(frutas)

Ahora la lista será ["manzana", "plátano", "uva"].

Ojo: aquí no estás añadiendo un nuevo elemento, estás sustituyendo uno que ya existía.

2. Acceder desde el final

Python también permite acceder desde el final de la lista usando índices negativos. Esto es muy útil cuando no quieres contar todas las posiciones a mano. 

numeros = [10, 20, 30, 40]
print(numeros[-1])
print(numeros[-2])
  • -1 es el último elemento
  • -2 es el penúltimo

Primer elemento

numeros[0]

Se usa muchísimo para acceder al inicio de la lista.

Último elemento

numeros[-1]

Es una de las formas más prácticas de trabajar con listas.

Ejemplo directo

numeros = [5, 8, 12, 20]
print(numeros[0])
print(numeros[-1])

Aquí obtienes el primer y el último valor de la lista.

Idea útil

Cuando trabajas con listas, preguntar por “el primero”, “el último” o “cambiar uno concreto” es de lo más habitual.

Idea práctica: si conoces bien los índices, puedes leer, cambiar y localizar datos dentro de una lista con mucha rapidez.

Obtener una parte de la lista

A veces no quieres un solo elemento, sino un tramo de la lista. Para eso puedes usar slicing. 

animales = ["gato", "perro", "loro", "pez"]
print(animales[1:3])

Esto devuelve ["perro", "loro"].

En slicing, se empieza en el primer índice y se llega solo hasta antes del segundo.

Cómo razonar si un índice es válido

Aquí es donde mucha gente se equivoca al principio.

Error típico: índice fuera de rango

frutas = ["manzana", "pera", "uva"]
print(frutas[3])

Este código da error porque esa lista tiene índices 0, 1 y 2. El índice 3 no existe.

Usar len() para pensar mejor

frutas = ["manzana", "pera", "uva"]
print(len(frutas))

Si la longitud es 3, los índices válidos van de 0 a 2.

Si una lista tiene longitud 3, su último índice positivo es 2.

Cómo pensar acceso y modificación antes de escribirlo

Paso 1

Decide si necesitas un valor concreto, el primero, el último o un tramo.

Paso 2

Comprueba si vas a leer un valor o a cambiarlo.

Paso 3

Piensa si el índice que vas a usar existe de verdad.

Paso 4

Si dudas, usa len() para razonar mejor la longitud y las posiciones.

Gran parte de los fallos aquí no vienen del código, sino de pensar mal qué posición estás usando.

Piensa antes de ejecutar

Antes de lanzar código, acostúmbrate a predecir qué hará cada línea.

Código

animales = ["gato", "perro", "loro", "pez"]
print(animales[0])
print(animales[-1])
print(animales[1:3])

Preguntas

  • ¿qué imprime cada línea?
  • ¿por qué el último valor no sale en el slicing?
Pista: en [1:3] se empieza en 1, pero se llega solo hasta antes de 3.

Reto Muy fácil

Tu misión

Crea una lista con 4 asignaturas, muestra la primera y la última, y cambia la segunda por otra nueva.

Resolver en playground

Pista

Usa [0] para el primero, [-1] para el último y recuerda que modificar no es lo mismo que añadir.

Mostrar solución 
asignaturas = ["Matemáticas", "Historia", "Física", "Lengua"]
print(asignaturas[0])
print(asignaturas[-1])
asignaturas[1] = "Programación"
print(asignaturas)

Errores frecuentes

Error 1

Pensar que el primer índice es 1.

Error 2

Usar índices fuera del rango.

Error 3

Confundir modificar un elemento con añadir uno nuevo.

Error 4

No entender que el slicing no incluye el límite final.
