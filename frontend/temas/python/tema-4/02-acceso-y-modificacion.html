Cuando una lista ya existe, el siguiente paso natural es saber entrar en ella y modificar partes concretas.

Acceder a un elemento

Cada elemento de una lista tiene una posición, llamada índice. En Python, los índices empiezan en 0. 

frutas = ["manzana", "pera", "uva"]
print(frutas[0])
print(frutas[1])
print(frutas[2])

Esto imprimirá manzana, pera y uva.

En Python, el primer elemento no es el 1. Es el 0.

La lógica de los índices

Al principio cuesta, pero aquí está una de las bases que más vas a usar.

¿Por qué empieza en 0?

Es una convención muy común en programación. Cuesta un poco al principio, pero después se vuelve natural. Debes asumirlo cuanto antes porque aparece constantemente.

Qué debes interiorizar

  • si una lista tiene 3 elementos, sus índices son 0, 1 y 2
  • la longitud no coincide con el último índice
  • el índice representa posición, no cantidad total

Trabajar con posiciones dentro de una lista

Una vez sabes acceder por índice, puedes modificar valores, mirar desde el final y recuperar elementos clave.

Modificar Índices negativos Uso frecuente

1. Modificar un elemento

Las listas en Python son mutables, lo que significa que sus elementos se pueden cambiar después de crear la lista. 

frutas = ["manzana", "pera", "uva"]
frutas[1] = "plátano"
print(frutas)

Ahora la lista será ["manzana", "plátano", "uva"].

Aquí no estás añadiendo un nuevo elemento, estás sustituyendo uno que ya existía.

2. Acceder desde el final

Python también permite acceder desde el final de la lista usando índices negativos. Esto es muy útil cuando no quieres contar todas las posiciones a mano. 

numeros = [10, 20, 30, 40]
print(numeros[-1])
print(numeros[-2])
  • -1 es el último elemento
  • -2 es el penúltimo

Primer elemento

numeros[0]

Se usa muchísimo para acceder al inicio de la lista.

Último elemento

numeros[-1]

Es una de las formas más prácticas de trabajar con listas.

Ejemplo directo

numeros = [5, 8, 12, 20]
print(numeros[0])
print(numeros[-1])

Aquí obtienes el primer y el último valor.

Idea útil

Los índices negativos te ahorran contar posiciones cuando solo te interesa el final.

Índices fuera de rango

numeros = [10, 20, 30]
print(numeros[3])

Esto falla porque la lista tiene 3 elementos, pero sus índices válidos son 0, 1 y 2.

Si dudas, razona primero con len(lista) antes de acceder a una posición.

Slicing: extraer una parte

numeros = [10, 20, 30, 40, 50]
print(numeros[1:4])

Esto devuelve [20, 30, 40].

En slicing, el límite final no se incluye.

Acceder, modificar y añadir no son lo mismo

Leer

lista[1] accede al valor.

Modificar

lista[1] = "nuevo" sustituye un valor existente.

Añadir

Para añadir nuevos elementos necesitarás métodos, no sustitución por índice.

Idea clave

No confundas “cambiar una posición” con “hacer crecer la lista”.

Errores frecuentes

Error 1

Pensar que el primer índice es 1.

Error 2

Usar índices fuera del rango.

Error 3

Confundir modificar un elemento con añadir uno nuevo.

Error 4

No entender que el slicing no incluye el límite final.

Buenas prácticas

Piensa en posiciones

Antes de escribir, identifica qué índice necesitas realmente.

Usa índices negativos con intención

Son muy útiles cuando necesitas el final sin contar toda la lista.

Comprueba el rango

Si dudas, razona con len() antes de ejecutar.

Diferencia acciones

No confundas leer, modificar y cortar una parte: son operaciones distintas.
