4.3 · Recorrer listas: aplicar bucles a colecciones de datos

¿Qué significa recorrer una lista?

Recorrer una lista significa pasar por sus elementos uno a uno para leerlos, mostrarlos, procesarlos, contarlos o transformarlos.

Esta es una habilidad central en programación.

Recorrer con for

frutas = ["manzana", "pera", "uva"]

for fruta in frutas:
    print(fruta)

En cada iteración, la variable fruta toma el siguiente elemento de la lista.

Ventaja de este enfoque

No necesitas pensar en índices para los casos más simples. Python te da directamente cada elemento.

Recorrer para acumular

Un patrón muy común es usar una lista y un acumulador. 

numeros = [10, 20, 30, 40]
suma = 0

for numero in numeros:
    suma = suma + numero

print(suma)

Aquí se recorre la lista para sumar todos sus elementos.

Recorrer para contar condiciones

numeros = [2, 5, 8, 11, 14]
contador_pares = 0

for numero in numeros:
    if numero % 2 == 0:
        contador_pares = contador_pares + 1

print(contador_pares)

Aquí no se suma el valor, se cuenta cuántos cumplen una condición.

Recorrer usando índices

A veces sí necesitas las posiciones. En esos casos puedes usar range(len(lista)). 

frutas = ["manzana", "pera", "uva"]

for i in range(len(frutas)):
    print(i, frutas[i])

Esto muestra el índice y el valor.

Cuándo recorrer con elemento y cuándo con índice

  • si solo necesitas el valor, usa for elemento in lista
  • si necesitas la posición, usa índices
No uses índices por costumbre si no los necesitas. Complica la lectura sin aportar valor.

Buscar un valor

nombres = ["Ana", "Luis", "Marta", "Carlos"]

for nombre in nombres:
    if nombre == "Marta":
        print("Encontrado")

También puedes combinar esto con break si quieres parar en cuanto lo encuentres.

💭 Piensa antes de ejecutar

numeros = [3, 6, 9]

for numero in numeros:
    print(numero * 2)
  • ¿qué se imprime en cada iteración?
  • ¿cuántas iteraciones hay?

⚠️ Error típico: confundir el elemento con el índice

frutas = ["manzana", "pera", "uva"]

for fruta in frutas:
    print(frutas[fruta])

Esto está mal, porque fruta contiene valores como "manzana", no índices numéricos.

Buenas prácticas

  • pon nombres correctos a la variable del bucle
  • si recorres notas, usa nota, no x
  • separa claramente acumuladores y contadores
  • elige la forma más simple posible

Mini-reto guiado

Recorre una lista de precios y muestra cada precio con IVA calculado al 21%. 

precios = [10, 20, 30]

for precio in precios:
    print(precio * 1.21)

Pregunta de entrevista

¿Cuándo usarías índices al recorrer una lista?

Respuesta sólida: cuando necesito conocer o utilizar la posición de cada elemento, no solo su valor.
