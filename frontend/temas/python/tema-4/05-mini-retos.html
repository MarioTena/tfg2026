Listas Bucles Métodos

Objetivo

Los problemas reales casi nunca usan una sola idea aislada. Aquí empiezas a combinar varias piezas a la vez.

Idea clave: no solo importa saber qué es una lista, sino qué puedes hacer con ella.

Cómo usar esta pantalla

  • lee el reto
  • entiende qué hace el código
  • pruébalo en el playground
  • cámbialo tú para comprobar que lo controlas
No memorices el código. Intenta explicar qué recorre el bucle, qué se modifica y qué se imprime.

Misiones rápidas

Mini-reto 1 · Suma de una lista

Recorre una lista y acumula el total.

numeros = [5, 10, 15, 20]
suma = 0

for numero in numeros:
    suma = suma + numero

print(suma)

Mini-reto 2 · Contar aprobados

Cuenta cuántas notas son mayores o iguales que 5.

notas = [3, 7, 5, 2, 9, 6]
contador = 0

for nota in notas:
    if nota >= 5:
        contador = contador + 1

print(contador)

Mini-reto 3 · Buscar un nombre

Recorre la lista y detente cuando encuentres el valor buscado.

nombres = ["Ana", "Luis", "Marta", "Carlos"]

for nombre in nombres:
    if nombre == "Marta":
        print("Encontrado")
        break

Mini-reto 4 · Modificar un elemento

Cambia un valor concreto usando su posición.

colores = ["rojo", "verde", "azul"]
colores[1] = "amarillo"
print(colores)

Mini-reto 5 · Lista ordenada

Usa un método para ordenar los elementos.

numeros = [8, 3, 1, 9, 4]
numeros.sort()
print(numeros)

Mini-reto 6 · Filtrar pares

Recorre la lista e imprime solo los valores que cumplen una condición.

numeros = [1, 2, 3, 4, 5, 6]

for numero in numeros:
    if numero % 2 == 0:
        print(numero)

Cómo pensar este tipo de ejercicios

Antes de escribir o modificar código, estructura el problema.

Paso a paso

  1. identifica qué representa la lista
  2. decide si necesitas valor, índice o ambos
  3. si hay que acumular o contar, crea la variable antes del bucle
  4. si hay que buscar, piensa si conviene usar break
  5. si modificas la lista, asegúrate de entender qué cambia exactamente

Regla útil

Cuando trabajas con listas, casi siempre estás haciendo una de estas cosas: recorrer, buscar, contar, modificar, añadir, eliminar u ordenar.
