4.4 · Métodos de listas: añadir, eliminar y organizar elementos

¿Qué es un método?

Un método es una acción que un objeto puede realizar. Las listas tienen varios métodos muy útiles.

append(): añadir al final

tareas = ["estudiar", "practicar"]
tareas.append("repasar")
print(tareas)

Ahora la lista tendrá un elemento más al final.

insert(): insertar en una posición

numeros = [10, 30, 40]
numeros.insert(1, 20)
print(numeros)

Inserta 20 en la posición 1.

remove(): eliminar por valor

frutas = ["manzana", "pera", "uva"]
frutas.remove("pera")
print(frutas)

Elimina la primera aparición de ese valor.

pop(): eliminar por posición

numeros = [10, 20, 30, 40]
numeros.pop(2)
print(numeros)

Elimina el elemento de índice 2. 

nombres = ["Ana", "Luis", "Marta"]
ultimo = nombres.pop()
print(ultimo)
print(nombres)

Si no indicas posición, elimina el último.

sort(): ordenar

numeros = [4, 1, 9, 2]
numeros.sort()
print(numeros)

Ordena la lista de menor a mayor.

reverse(): invertir el orden

numeros = [1, 2, 3, 4]
numeros.reverse()
print(numeros)

count(): contar apariciones

numeros = [1, 2, 2, 3, 2]
print(numeros.count(2))

Devuelve cuántas veces aparece ese valor.

index(): encontrar la posición

frutas = ["manzana", "pera", "uva"]
print(frutas.index("pera"))

Devuelve el índice de la primera aparición.

⚠️ Error típico: pensar que un método crea otra lista automáticamente

Muchos métodos modifican la lista original. No siempre devuelven una nueva. 

numeros = [3, 1, 2]
numeros.sort()
print(numeros)

💭 Piensa antes de ejecutar

datos = [10, 20, 30]
datos.append(40)
datos.remove(20)
print(datos)
  • ¿qué lista queda al final?
  • ¿en qué orden se aplican los cambios?

Errores frecuentes

  • usar remove() pensando en índices
  • usar pop() pensando en valores
  • olvidar que algunos métodos modifican la lista original
  • intentar eliminar valores que no existen

Mini-reto guiado

Crea una lista de compras, añade dos productos, elimina uno y ordénala alfabéticamente. 

compras = ["pan", "leche"]
compras.append("huevos")
compras.append("arroz")
compras.remove("leche")
compras.sort()
print(compras)

Pregunta de entrevista

¿Qué diferencia hay entre remove() y pop()?

Respuesta sólida: remove() elimina por valor; pop() elimina por posición, y si no se indica ninguna, elimina el último elemento.
