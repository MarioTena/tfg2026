Qué vas a dominar hoy

Las listas no solo guardan elementos. También ofrecen métodos para modificarlas y consultar información sin tener que escribir todo manualmente.

Añadir Eliminar Organizar

¿Qué es un método?

Un método es una acción que un objeto puede realizar. Las listas tienen varios métodos muy útiles.

Idea clave: un método te permite hacer algo con la lista sin tener que construir esa lógica desde cero.

Qué debes ser capaz de hacer

  • añadir elementos
  • insertarlos en una posición concreta
  • eliminar por valor o por posición
  • ordenar o invertir una lista
  • contar apariciones y localizar posiciones

Dónde aparecen los métodos de listas en problemas reales

Compras

Añadir y quitar productos de un carrito.

Tareas

Actualizar una lista de pendientes según lo que se completa o cambia.

Ordenación

Reorganizar datos alfabéticamente o por valor.

Consultas

Saber cuántas veces aparece algo o dónde está por primera vez.

Cuando una lista ya existe y quieres manipularla, los métodos aparecen constantemente.

Métodos para añadir elementos

No siempre quieres reemplazar un valor. A veces necesitas ampliar la lista.

append() · añadir al final

tareas = ["estudiar", "practicar"]
tareas.append("repasar")
print(tareas)

Ahora la lista tendrá un elemento más al final.

Es una de las formas más comunes de ampliar una lista.

insert() · insertar en una posición

numeros = [10, 30, 40]
numeros.insert(1, 20)
print(numeros)

Inserta 20 en la posición 1.

A diferencia de append(), aquí decides en qué lugar entra el nuevo elemento.

Métodos para eliminar elementos

Aquí es muy importante distinguir entre eliminar por valor y eliminar por posición.

remove() · eliminar por valor

frutas = ["manzana", "pera", "uva"]
frutas.remove("pera")
print(frutas)

Elimina la primera aparición de ese valor.

Aquí no le pasas un índice: le pasas el valor que quieres quitar.

pop() · eliminar por posición

numeros = [10, 20, 30, 40]
numeros.pop(2)
print(numeros)

Elimina el elemento de índice 2. 

nombres = ["Ana", "Luis", "Marta"]
ultimo = nombres.pop()
print(ultimo)
print(nombres)

Si no indicas posición, elimina el último.

Regla importante: remove() piensa en valores; pop() piensa en posiciones.

Métodos para organizar y consultar

Una vez tienes datos dentro de la lista, muchas veces necesitas ordenarlos o sacar información sobre ellos.

sort() · ordenar

numeros = [4, 1, 9, 2]
numeros.sort()
print(numeros)

Ordena la lista de menor a mayor.

reverse() · invertir

numeros = [1, 2, 3, 4]
numeros.reverse()
print(numeros)

Invierte el orden actual de la lista.

count() · contar apariciones

numeros = [1, 2, 2, 3, 2]
print(numeros.count(2))

Devuelve cuántas veces aparece ese valor.

index() · encontrar la posición

frutas = ["manzana", "pera", "uva"]
print(frutas.index("pera"))

Devuelve el índice de la primera aparición.

Cómo pensar un método antes de usarlo

Paso 1

Pregúntate si quieres añadir, quitar, ordenar o consultar.

Paso 2

Decide si vas a trabajar con un valor o con una posición.

Paso 3

Comprueba si el método modifica la lista original.

Paso 4

Piensa qué pasará con la lista después de aplicar el método.

Mucha confusión con los métodos viene de no distinguir bien si actúan sobre valores, índices o sobre la lista completa.

Error típico: pensar que un método crea otra lista automáticamente

Muchos métodos modifican la lista original. No siempre devuelven una nueva. 

numeros = [3, 1, 2]
numeros.sort()
print(numeros)
Cuando uses métodos de listas, pregúntate siempre: “¿esto cambia la lista original o me devuelve otra cosa?”

Piensa antes de ejecutar

Observa cómo varios métodos pueden encadenar cambios sobre la misma lista.

Código

datos = [10, 20, 30]
datos.append(40)
datos.remove(20)
print(datos)

Preguntas

  • ¿qué lista queda al final?
  • ¿en qué orden se aplican los cambios?
Pista: primero se añade el 40 y después se elimina el 20.

Mini-reto guiado

Ahora toca usar varios métodos dentro de una misma situación.

Reto Muy fácil

Tu misión

Crea una lista de compras, añade dos productos, elimina uno y ordénala alfabéticamente.

Resolver en playground

Pista

Aquí vas a combinar varios métodos seguidos. Piensa qué hace cada uno sobre la misma lista.

Mostrar solución 
compras = ["pan", "leche"]
compras.append("huevos")
compras.append("arroz")
compras.remove("leche")
compras.sort()
print(compras)

Errores frecuentes

Error 1

Usar remove() pensando en índices.

Error 2

Usar pop() pensando en valores.

Error 3

Olvidar que algunos métodos modifican la lista original.

Error 4

Intentar eliminar valores que no existen.

Buenas prácticas

Piensa en la intención

Antes de elegir un método, define bien qué quieres hacer con la lista.

Valor o posición

Diferencia siempre si estás trabajando con el contenido o con el índice.

Comprueba el efecto

Fíjate si el método cambia la lista original o no.

Haz pruebas pequeñas

Mientras aprendes, prueba los métodos con listas cortas para ver su efecto real.
