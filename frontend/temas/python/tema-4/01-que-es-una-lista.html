Qué vas a entender hoy

Una lista permite guardar varios valores relacionados dentro de una sola variable, manteniendo el orden.

Objetivo Base para bucles

Qué problema resuelve

Hasta ahora has usado variables sueltas: una para el nombre, otra para la edad, otra para la nota. Pero en programación real muchas veces necesitas trabajar con varios datos del mismo grupo.

Ejemplo real

Los nombres de varios alumnos.

Ejemplo real

Los productos de un pedido.

Ejemplo real

Las temperaturas de varios días.

Ejemplo real

Los mensajes de un chat.

Idea clave: una lista agrupa varios valores relacionados dentro de una sola estructura.

Qué debes recordar

  • se escribe entre []
  • los elementos van separados por comas
  • mantiene el orden
  • puede guardar varios elementos
Piensa así:
una lista no es solo “muchos datos juntos”, sino “muchos datos juntos y ordenados”.

Ejemplo rápido

Lista de números

numeros = [10, 20, 30, 40]

Lista de nombres

nombres = ["Ana", "Luis", "Marta"]

Lista vacía

tareas = []

Lista con mezcla

mixta = ["Ana", 25, True]

Piensa antes de ejecutar

Antes de probar código, acostúmbrate a predecir qué va a pasar.

Código

colores = ["rojo", "verde", "azul"]
print(colores)
print(len(colores))

Preguntas

  • ¿Qué se imprime en la primera línea?
  • ¿Qué devuelve len(colores)?
Intenta responder antes de ir al playground.

Pista

Empieza creando una variable llamada ciudades y guarda dentro varios nombres entre corchetes.

Mostrar solución 
ciudades = ["Madrid", "Sevilla", "Bilbao", "Valencia", "Málaga"]
print(ciudades)
print(len(ciudades))

Conceptos clave del tema

El orden importa

No es lo mismo [10, 20, 30] que [30, 20, 10]. En una lista, la posición cambia el significado.

Longitud

Puedes saber cuántos elementos tiene una lista usando len().

Uso real

Las listas se usan muchísimo porque permiten recorrer, añadir, modificar y eliminar elementos.

Buena práctica

Usa nombres claros como nombres, precios o notas.

Errores frecuentes

Error 1

Pensar que una lista es una sola cadena de texto.

Error 2

Olvidar las comas entre elementos.

Error 3

Confundir corchetes [] con paréntesis ().

Error 4

No entender que el orden de los elementos importa.

Pregunta típica de entrevista

Pregunta: ¿Qué es una lista en Python?

Respuesta sólida: una lista es una estructura de datos ordenada que permite guardar varios elementos en una sola variable.
