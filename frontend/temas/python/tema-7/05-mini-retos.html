¿Devuelven exactamente lo mismo? ¿Qué diferencia conceptual hay entre pedir una posición y pedir un fragmento?

Reto 2 · Límite del slicing

Código

texto = "programa"
print(texto[:4])

Pregunta

¿Por qué se extraen cuatro caracteres y no cinco? ¿Qué significa realmente ese 4 en el slicing?

En slicing, el final marca hasta dónde llega el corte sin incluir esa posición.

Reto 3 · Inmutabilidad real

Código

texto = "HOLA"
texto.lower()
print(texto)

Pregunta

¿Por qué sigue imprimiendo HOLA? ¿Qué olvidó hacer el programador?

Este es uno de los errores más importantes del tema: pensar que un método cambia el string original sin guardar nada.

Método rápido para comprobar una operación sobre strings

Qué hacer antes de dar un ejercicio por bien resuelto.

Paso a paso

  1. decide si quieres un carácter, un fragmento, una palabra o toda la cadena
  2. piensa si el string debe transformarse antes
  3. comprueba si la operación devuelve string, lista, booleano o entero
  4. pregúntate si necesitas guardar el resultado
  5. elige entre método o bucle según la tarea

Regla útil

Si no sabes qué tipo de resultado produce una operación, todavía no la dominas del todo.
