Texto Slicing Métodos

Objetivo

Vas a practicar cómo transformar, limpiar, dividir y analizar texto según el problema.

Idea clave: no siempre trabajas con texto entero; a veces trabajas con caracteres, otras con palabras y otras con fragmentos.

Cómo usar esta pantalla

  • lee el reto
  • decide si necesitas carácter, palabra o fragmento
  • identifica si te conviene un método o un bucle
  • pruébalo en el playground y cambia textos
Antes de escribir, pregúntate: ¿quiero extraer, transformar, contar o dividir el texto?

Misiones rápidas

Mini-reto 1 · Primera y última letra

Accede a posiciones concretas del texto.

palabra = "programa"
print(palabra[0])
print(palabra[-1])

Mini-reto 2 · Texto invertido

Usa slicing para cambiar el orden.

texto = "python"
print(texto[::-1])

Mini-reto 3 · Normalizar nombre

Limpia espacios y unifica mayúsculas/minúsculas.

nombre = "   ANA   "
nombre = nombre.strip().lower()
print(nombre)

Mini-reto 4 · Contar vocales

Combina recorrido y condición sobre texto.

texto = "Programacion"
contador = 0

for letra in texto.lower():
    if letra in "aeiou":
        contador = contador + 1

print(contador)

Mini-reto 5 · Separar palabras

Divide una frase en una lista de palabras.

frase = "Python es muy util"
palabras = frase.split()
print(palabras)

Mini-reto 6 · Reemplazar texto

Transforma una parte concreta del mensaje.

frase = "Me gusta Java"
frase = frase.replace("Java", "Python")
print(frase)

Cómo pensar este tipo de ejercicios

Antes de escribir, decide bien el nivel al que vas a trabajar el texto.

Paso a paso

  1. decide si trabajas a nivel de carácter o de palabra
  2. si el texto es irregular, normalízalo primero
  3. si necesitas contar, crea un acumulador
  4. si necesitas extraer una parte, piensa en slicing
  5. si necesitas transformar, revisa si ya existe un método adecuado

Regla útil

Si Python ya trae un método que resuelve parte del problema, úsalo antes de complicarte con lógica innecesaria.

Pregunta típica de entrevista

Pregunta: ¿Qué diferencia hay entre recorrer un string carácter a carácter y usar split()?

Respuesta sólida: recorrer un string trabaja a nivel de caracteres; split() lo divide en partes, normalmente palabras, y permite analizarlo a otro nivel.
