Qué vas a entender hoy

Un string no solo se puede leer o cortar: también puedes limpiarlo, transformarlo, buscar dentro de él y convertirlo en partes más manejables.

Objetivo Transformar texto

Qué problema resuelve

En programas reales los textos rara vez llegan perfectos. A veces tienen espacios de sobra, mayúsculas inconsistentes o partes que necesitas sustituir, separar o validar.

Normalizar

Pasar todo a mayúsculas o minúsculas.

Limpiar

Eliminar espacios sobrantes al principio o al final.

Sustituir

Cambiar una parte del texto por otra.

Separar

Convertir una frase en varias partes con split().

Idea clave: los métodos de strings te permiten trabajar con texto sin tener que escribir toda la lógica desde cero.

Qué debes vigilar

  • los strings son inmutables
  • muchos métodos devuelven un string nuevo
  • si no guardas el resultado, el cambio se pierde
  • algunos métodos transforman y otros analizan
Piensa así:
aplicar un método a un string no suele “retocarlo por dentro”, sino producir una nueva versión del texto.

Por qué importan los métodos de strings

Los strings aparecen en casi todos los programas: nombres, mensajes, formularios, búsquedas o validaciones. Por eso Python ofrece muchos métodos para trabajar con ellos de forma práctica.

lower() y upper()

texto = "PyThOn"
print(texto.lower())
print(texto.upper())

Son muy útiles para comparar textos sin depender de mayúsculas y minúsculas.

Si dos textos deberían considerarse iguales aunque cambien las mayúsculas, normalizarlos antes suele ser buena idea.

strip()

texto = "   hola   "
print(texto.strip())

Elimina espacios al principio y al final. Es muy útil cuando trabajas con entrada de usuario.

replace()

texto = "Hola mundo"
print(texto.replace("mundo", "Python"))

Sustituye una parte del texto por otra.

find()

texto = "programacion"
print(texto.find("gra"))

Devuelve la posición donde empieza la subcadena, o -1 si no la encuentra.

startswith() y endswith()

archivo = "datos.csv"
print(archivo.startswith("da"))
print(archivo.endswith(".csv"))

Muy útiles para comprobaciones simples, por ejemplo validar extensiones o prefijos.

split() y join()

split()

frase = "uno dos tres"
print(frase.split())

Convierte un string en una lista separándolo normalmente por espacios.

join()

palabras = ["Hola", "mundo"]
print(" ".join(palabras))

Une los elementos de una lista en un único string.

Regla mental: split() separa un texto; join() une varias partes en un texto.

Importante: los strings son inmutables

Métodos como lower() o replace() no cambian el string original; devuelven uno nuevo. 

texto = "HOLA"
texto.lower()
print(texto)

Esto seguirá imprimiendo HOLA. Si quieres conservar el cambio, debes guardarlo. 

texto = texto.lower()

Cómo pensar qué método necesitas

Paso 1

Pregúntate si quieres transformar, limpiar, buscar o separar el texto.

Paso 2

Elige el método que resuelva exactamente esa tarea.

Paso 3

Comprueba si el método devuelve un nuevo string.

Paso 4

Guarda el resultado si quieres seguir usándolo después.

Piensa antes de ejecutar

Antes de probar el código, intenta seguir qué hace cada método sobre el texto.

Código

texto = "  PyThOn  "
texto = texto.strip().lower()
print(texto)

Mini-reto guiado

Ahora toca limpiar y transformar texto con intención.

Reto Muy fácil

Tu misión

Crea una frase con espacios sobrantes y conviértela a minúsculas sin espacios laterales.

Resolver en playground

Pista

Primero limpia los espacios y después normaliza el texto.

Mostrar solución 
frase = "   HOLA PYTHON   "
frase = frase.strip().lower()
print(frase)

Errores frecuentes

Error 1

Pensar que los métodos modifican el string original.

Error 2

Olvidar guardar el resultado.

Error 3

Usar find() sin considerar el caso -1.

Error 4

No distinguir entre split() y join().

Buenas prácticas

Normaliza

Si vas a comparar texto, considera usar lower().

Limpia entradas

strip() suele ser útil con texto escrito por usuarios.

Guarda cambios

Recuerda que muchos métodos devuelven una nueva versión del string.

Elige bien

No uses varios métodos por costumbre; usa el que resuelva la tarea concreta.
