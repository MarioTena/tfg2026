7.4 · Recorrer y analizar texto: aplicar bucles y lógica sobre strings

Recorrer un string con for

texto = "Python"

for letra in texto:
    print(letra)

En cada iteración, la variable letra toma un carácter distinto.

Contar caracteres concretos

texto = "banana"
contador = 0

for letra in texto:
    if letra == "a":
        contador = contador + 1

print(contador)

Esto cuenta cuántas veces aparece la letra a.

Buscar si existe un carácter o subcadena

texto = "programacion"

if "gra" in texto:
    print("Encontrado")

El operador in también funciona con strings.

Contar vocales

texto = "python es util"
contador = 0

for letra in texto:
    if letra in "aeiou":
        contador = contador + 1

print(contador)

Este tipo de ejercicio es muy frecuente para aprender a combinar texto, bucles y condicionales.

Normalizar antes de analizar

A veces conviene transformar el texto antes de procesarlo. Por ejemplo, pasar todo a minúsculas. 

texto = "PyThOn"
texto = texto.lower()

for letra in texto:
    print(letra)

Recorrer con índices

texto = "hola"

for i in range(len(texto)):
    print(i, texto[i])

Esto es útil cuando necesitas tanto posición como carácter.

Separar palabras y analizarlas

frase = "Python es muy potente"
palabras = frase.split()

for palabra in palabras:
    print(palabra)

Así pasas de trabajar a nivel de carácter a trabajar a nivel de palabra.

💭 Piensa antes de ejecutar

texto = "casa"
contador = 0

for letra in texto:
    if letra == "a":
        contador = contador + 1

print(contador)
  • ¿cuántas veces aparece la letra a?
  • ¿qué ocurriría si el texto fuera "Casa"?

Errores frecuentes

  • no normalizar mayúsculas/minúsculas antes de comparar
  • confundir contar caracteres con contar palabras
  • usar índices cuando no hacen falta
  • olvidar inicializar el contador antes del bucle

Mini-reto guiado

Cuenta cuántas letras e hay en una frase. 

frase = "Este tema es interesante"
contador = 0

for letra in frase.lower():
    if letra == "e":
        contador = contador + 1

print(contador)

Pregunta de entrevista

¿Por qué puede ser útil convertir un texto a minúsculas antes de analizarlo?

Respuesta sólida: porque así evitas que mayúsculas y minúsculas se traten como caracteres distintos cuando en la lógica del problema deberían considerarse iguales.
