Qué vas a entender hoy

Un string puede recorrerse carácter a carácter para contar letras, buscar patrones simples o extraer información útil del texto.

Objetivo Texto + bucles

Qué problema resuelve

Muchas tareas reales con texto consisten en analizarlo: contar apariciones, comprobar si contiene algo, normalizarlo o recorrerlo para estudiar cada carácter.

Contar

Calcular cuántas veces aparece una letra o grupo de letras.

Buscar

Comprobar si una subcadena existe dentro del texto.

Normalizar

Preparar el texto antes de compararlo o analizarlo.

Recorrer

Procesar carácter a carácter o palabra a palabra.

Idea clave: cuando recorres un string, cada iteración trabaja con una pequeña parte del texto.

Qué debes vigilar

  • no confundir carácter con palabra
  • normalizar mayúsculas y minúsculas si hace falta
  • inicializar bien contadores antes del bucle
  • usar índices solo cuando realmente los necesites
Piensa así: analizar texto consiste en dividir un problema grande en pequeñas comprobaciones repetidas.

Dónde aparece recorrer y analizar texto en problemas reales

Contadores

Calcular cuántas veces aparece una letra o una palabra.

Búsqueda

Comprobar si un texto contiene un patrón o carácter concreto.

Normalización

Preparar el texto antes de compararlo o validarlo.

Procesamiento

Recorrer carácter a carácter para analizar o transformar información.

Recorrer y analizar texto aparece cuando necesitas dividir un problema textual en comprobaciones pequeñas y repetidas.

Recorrer un string con for

texto = "Python"

for letra in texto:
    print(letra)

En cada iteración, la variable letra toma un carácter distinto.

Contar caracteres concretos

texto = "banana"
contador = 0

for letra in texto:
    if letra == "a":
        contador = contador + 1

print(contador)

Esto cuenta cuántas veces aparece la letra a.

Buscar si existe un carácter o subcadena

texto = "programacion"

if "gra" in texto:
    print("Encontrado")

El operador in también funciona con strings y sirve para búsquedas simples.

Contar vocales

texto = "python es util"
contador = 0

for letra in texto:
    if letra in "aeiou":
        contador = contador + 1

print(contador)

Este tipo de ejercicio es muy frecuente para aprender a combinar texto, bucles y condicionales.

Normalizar antes de analizar

A veces conviene transformar el texto antes de procesarlo, por ejemplo pasarlo todo a minúsculas. 

texto = "PyThOn"
texto = texto.lower()

for letra in texto:
    print(letra)
Si el análisis no debería distinguir mayúsculas de minúsculas, normaliza primero el texto.

Recorrer con índices

texto = "hola"

for i in range(len(texto)):
    print(i, texto[i])

Esto es útil cuando necesitas tanto la posición como el carácter.

Separar palabras y analizarlas

frase = "Python es muy potente"
palabras = frase.split()

for palabra in palabras:
    print(palabra)

Así pasas de trabajar a nivel de carácter a trabajar a nivel de palabra.

Carácter, palabra o subcadena: decide el nivel

Carácter

Cuando analizas letra a letra, como en conteos simples.

Palabra

Cuando primero separas con split().

Subcadena

Cuando buscas grupos de caracteres como "gra" o ".csv".

Idea útil

Antes de escribir, decide bien a qué nivel quieres trabajar el texto.

Método o bucle: ¿qué conviene?

Método

Cuando Python ya tiene una herramienta clara para esa tarea, como split() o count().

Bucle

Cuando necesitas una lógica más personalizada o varias comprobaciones a la vez.

Criterio

No siempre hay una única forma correcta, pero sí una más cómoda o más clara.

Pregunta útil

¿Ya existe un método que haga esto, o necesito controlar el proceso paso a paso?

Casos reales de análisis

Contar

Cuántas vocales hay en una frase.

Validar

Si un archivo termina en .csv.

Buscar

Si una palabra o fragmento aparece en el texto.

Separar

Cuántas palabras contiene una frase.

Cómo pensar un análisis de texto antes de escribirlo

Paso 1

Decide si vas a analizar caracteres, palabras o subcadenas.

Paso 2

Piensa si debes normalizar el texto antes.

Paso 3

Prepara contadores o acumuladores antes del bucle.

Paso 4

Dentro del bucle, haz una comprobación simple y clara.

Piensa antes de ejecutar

Antes de probar el código, sigue mentalmente el recorrido del bucle y el cambio del contador.

Código

texto = "casa"
contador = 0

for letra in texto:
    if letra == "a":
        contador = contador + 1

print(contador)

Preguntas

  • ¿cuántas veces aparece la letra a?
  • ¿en qué iteraciones cambia el contador?
  • ¿qué pasaría si normalizaras el texto antes?
