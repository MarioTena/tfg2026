Qué vas a entender hoy

Si un string es una secuencia, entonces puedes acceder a posiciones concretas y extraer partes del texto igual que hacías con otras estructuras.

Objetivo Slicing

Qué problema resuelve

Muchas veces no necesitas todo el texto, sino una letra concreta, el último carácter o un fragmento. La indexación y el slicing sirven para eso.

Idea clave: acceder a una posición y extraer un fragmento no son exactamente la misma operación.

Qué debes vigilar

  • los índices empiezan en 0
  • los índices negativos cuentan desde el final
  • el slicing no incluye el límite final
  • un índice fuera de rango provoca error
Piensa así:
acceder es pedir una posición exacta; hacer slicing es pedir un tramo del texto.

Acceder a una posición concreta

Igual que en las listas, los strings usan índices que empiezan en 0. 

texto = "Python"
print(texto[0])
print(texto[1])

Esto imprimirá P y y.

Índices negativos

También puedes acceder desde el final. 

texto = "Python"
print(texto[-1])
print(texto[-2])
  • -1 es el último carácter
  • -2 es el penúltimo

Error típico: índice fuera de rango

texto = "sol"
print(texto[3])

Esto da error porque los índices válidos son 0, 1 y 2.

Slicing: extraer una parte del string

El slicing permite tomar un fragmento de texto. 

texto = "Python"
print(texto[1:4])

Devuelve yth. Empieza en el índice 1 y llega hasta antes del 4.

Slicing desde el inicio o hasta el final

texto = "Python"
print(texto[:3])
print(texto[3:])

Esto devuelve Pyt y hon.

Copiar un string completo

texto = "Python"
print(texto[:])

Devuelve todo el string.

Salto en slicing

texto = "Python"
print(texto[0:6:2])

Va tomando caracteres de dos en dos.

Invertir un string

texto = "Python"
print(texto[::-1])

Esto devuelve el texto al revés.

Cómo pensar indexación y slicing antes de escribirlos

Paso 1

Decide si quieres un carácter o un fragmento.

Paso 2

Si quieres una posición exacta, usa un índice.

Paso 3

Si quieres un tramo, usa slicing.

Paso 4

Recuerda que el límite final del slicing no se incluye.

Piensa antes de ejecutar

Antes de lanzar el código, sigue mentalmente qué devuelve cada acceso.

Código

palabra = "programa"
print(palabra[0])
print(palabra[-1])
print(palabra[2:6])

Preguntas

  • ¿qué devuelve cada línea?
  • ¿por qué el límite final no se incluye?
Pista: [2:6] empieza en 2 y llega solo hasta antes de 6.

Tu misión

Resolver en playground

Pista

La primera y la última letra se obtienen con índices; el fragmento inicial se obtiene con slicing.

Mostrar solución 
nombre = "Carlos"
print(nombre[0])
print(nombre[-1])
print(nombre[:3])

Errores frecuentes

Error 1

Pensar que el primer índice es 1.

Error 2

Olvidar que el slicing no incluye el final.

Error 3

Usar índices fuera de rango.

Error 4

Confundir un carácter con una subcadena.
