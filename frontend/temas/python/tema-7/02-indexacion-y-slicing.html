7.2 · Indexación y slicing: acceder a letras y fragmentos de texto

Acceder a una posición concreta

Igual que en las listas, los strings usan índices que empiezan en 0. 

texto = "Python"
print(texto[0])
print(texto[1])

Esto imprimirá P y y.

Índices negativos

También puedes acceder desde el final. 

texto = "Python"
print(texto[-1])
print(texto[-2])
  • -1 es el último carácter
  • -2 es el penúltimo

⚠️ Error típico: índice fuera de rango

texto = "sol"
print(texto[3])

Esto da error porque los índices válidos son 0, 1 y 2.

Slicing: extraer una parte del string

El slicing permite tomar un fragmento de texto. 

texto = "Python"
print(texto[1:4])

Devuelve yth. Empieza en el índice 1 y llega hasta antes del 4.

Slicing desde el inicio o hasta el final

texto = "Python"
print(texto[:3])
print(texto[3:])

Esto devuelve Pyt y hon.

Copiar un string completo

texto = "Python"
print(texto[:])

Devuelve todo el string.

Salto en slicing

texto = "Python"
print(texto[0:6:2])

Va tomando caracteres de dos en dos.

Invertir un string

texto = "Python"
print(texto[::-1])

Esto devuelve el texto al revés.

💭 Piensa antes de ejecutar

palabra = "programa"
print(palabra[0])
print(palabra[-1])
print(palabra[2:6])
  • ¿qué devuelve cada línea?
  • ¿por qué el límite final no se incluye?

Errores frecuentes

  • pensar que el primer índice es 1
  • olvidar que el slicing no incluye el final
  • usar índices fuera de rango
  • confundir un carácter con una subcadena

Mini-reto guiado

Guarda tu nombre en una variable y muestra:

  • la primera letra
  • la última letra
  • las tres primeras letras
nombre = "Carlos"
print(nombre[0])
print(nombre[-1])
print(nombre[:3])

Pregunta de entrevista

¿Qué diferencia hay entre texto[2] y texto[2:3]?

Respuesta sólida: ambos apuntan a la misma posición, pero el primero devuelve un carácter y el segundo devuelve una subcadena de longitud 1.
