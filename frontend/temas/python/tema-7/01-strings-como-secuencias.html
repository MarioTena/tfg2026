7.1 · Strings como secuencias: entender el texto como una estructura recorrible

Qué cambia respecto al tema 1

En el tema 1 viste que un string es texto, por ejemplo "Hola". Ahora vamos a dar un paso más importante: en Python, un string no es solo texto, sino también una secuencia de caracteres.

Pensar un string como secuencia permite recorrerlo, acceder a posiciones concretas, cortar partes y analizarlo.

Qué significa “secuencia”

Una secuencia es una estructura ordenada de elementos. En una lista los elementos pueden ser números o textos. En un string, los elementos son caracteres. 

texto = "Python"
print(texto)

Ese string está formado por los caracteres P, y, t, h, o y n.

Longitud de un string

Igual que con listas, puedes saber cuántos elementos tiene con len(). 

texto = "Python"
print(len(texto))

El resultado será 6.

Los espacios también cuentan

frase = "Hola mundo"
print(len(frase))

El espacio entre palabras también es un carácter. Esto es importante al contar, recorrer y analizar texto.

Un string es inmutable

Igual que ocurre con otros tipos de datos, un string no se modifica carácter a carácter de forma directa. Se considera inmutable.

Eso significa que, cuando parece que lo cambias, en realidad normalmente estás creando uno nuevo.

Diferencia con una lista

Ambos son secuencias, pero hay una diferencia muy importante:

  • una lista es mutable
  • un string es inmutable

Esta diferencia aparecerá mucho al programar.

Cuándo es importante ver un string como secuencia

  • cuando quieres contar letras
  • cuando quieres recorrer carácter a carácter
  • cuando quieres extraer una parte del texto
  • cuando necesitas validar formato
  • cuando buscas patrones simples

💭 Piensa antes de ejecutar

palabra = "sol"
print(palabra)
print(len(palabra))
  • ¿qué representa len(palabra)?
  • ¿qué pasaría si la palabra fuera "sol brillante"?

Errores frecuentes

  • pensar que un string es indivisible
  • olvidar que los espacios cuentan
  • no distinguir entre string y lista
  • no entender la idea de inmutabilidad

Mini-reto guiado

Crea una variable frase, guárdale una frase corta y muestra su longitud. 

frase = "Aprender Python"
print(frase)
print(len(frase))

Pregunta de entrevista

¿Por qué puede decirse que un string es una secuencia?

Respuesta sólida: porque está formado por caracteres ordenados sobre los que se puede acceder, recorrer y operar como elementos individuales.
