Un string no es solo texto. En Python también es una secuencia ordenada de caracteres sobre la que puedes contar, recorrer y operar.

Qué cambia respecto al tema 1

En el tema 1 viste que un string es texto, por ejemplo "Hola". Ahora vas a dar un paso más importante: en Python, un string no es solo texto, sino también una secuencia de caracteres.

Pensar un string como secuencia permite recorrerlo, acceder a posiciones concretas, cortar partes y analizarlo.

Qué significa “secuencia”

Una secuencia es una estructura ordenada de elementos. En una lista los elementos pueden ser números, booleanos o textos. En un string, los elementos son caracteres. 

texto = "Python"
print(texto)

Ese string está formado por los caracteres P, y, t, h, o y n.

Strings y listas: similitudes y diferencias

Lista

Secuencia de elementos generales.

String

Secuencia de caracteres.

Orden

En ambos casos, el orden importa.

Diferencia clave

El string representa texto; la lista puede representar muchas otras cosas.

Longitud de un string

Igual que con listas, puedes saber cuántos elementos tiene con len(). 

texto = "Python"
print(len(texto))

El resultado será 6.

Los espacios también cuentan

frase = "Hola mundo"
print(len(frase))

El espacio entre palabras también es un carácter. Esto es importante al contar, recorrer y analizar texto.

No todos los caracteres son letras

Letras

"Python"

Espacios

"Hola mundo"

Números

"Aula 3"

Signos

"Hola!"

Todo eso sigue siendo texto y todo cuenta como parte de la secuencia.

Casos simples importantes

String vacío

texto = ""

Un solo carácter

texto = "A"

Con espacios

texto = " a "

Idea útil

Un string puede ser muy corto, muy largo o incluso estar vacío.

Un string es inmutable

Igual que ocurre con otros tipos de datos, un string no se modifica carácter a carácter de forma directa. Se considera inmutable.

Eso significa que, cuando parece que lo cambias, en realidad normalmente estás creando uno nuevo.

Por qué importa la inmutabilidad de verdad

texto = "Hola"
# texto[0] = "h"

Esa idea no funciona en strings. No puedes cambiar una posición concreta como harías en una lista.

En strings, muchas operaciones no “retocan” el original: crean una nueva versión del texto.
