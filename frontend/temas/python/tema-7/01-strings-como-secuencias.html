Qué vas a entender hoy

Un string no es solo texto. En Python también es una secuencia ordenada de caracteres sobre la que puedes contar, recorrer y operar.

Qué cambia respecto al tema 1

En el tema 1 viste que un string es texto, por ejemplo "Hola". Ahora vas a dar un paso más importante: en Python, un string no es solo texto, sino también una secuencia de caracteres.

Pensar un string como secuencia permite recorrerlo, acceder a posiciones concretas, cortar partes y analizarlo.

Qué significa “secuencia”

Una secuencia es una estructura ordenada de elementos. En una lista los elementos pueden ser números o textos. En un string, los elementos son caracteres. 

texto = "Python"
print(texto)

Ese string está formado por los caracteres P, y, t, h, o y n.

Longitud de un string

Igual que con listas, puedes saber cuántos elementos tiene con len(). 

texto = "Python"
print(len(texto))

El resultado será 6.

Los espacios también cuentan

frase = "Hola mundo"
print(len(frase))

El espacio entre palabras también es un carácter. Esto es importante al contar, recorrer y analizar texto.

Un string es inmutable

Igual que ocurre con otros tipos de datos, un string no se modifica carácter a carácter de forma directa. Se considera inmutable.

Eso significa que, cuando parece que lo cambias, en realidad normalmente estás creando uno nuevo.

Diferencia con una lista

Lista

Es una secuencia mutable.

String

Es una secuencia inmutable.

Orden

En ambos casos el orden de los elementos importa.

Uso

Uno guarda elementos generales y el otro guarda caracteres.

Paso 2

Paso 4

Piensa antes de ejecutar

Antes de probar el código, piensa qué representa la longitud de un texto.

Código

palabra = "sol"
print(palabra)
print(len(palabra))

Preguntas

  • ¿qué representa len(palabra)?
  • ¿qué pasaría si la palabra fuera "sol brillante"?
Pista: la longitud cuenta todos los caracteres, incluidos los espacios.

Mini-reto guiado

Resolver en playground

Pista

Mostrar solución

Errores frecuentes

Error 1

Pensar que un string es indivisible.

Error 2

Olvidar que los espacios cuentan.

Error 3

No distinguir entre string y lista.

Error 4

No entender la idea de inmutabilidad.
