Qué entrenas aquí

En este proyecto empiezas a trabajar con información más rica y organizada. Ya no basta con guardar textos sueltos: ahora debes relacionar nombres y notas.

Datos Búsqueda Integración

Objetivo

Vas a crear un programa que permita registrar alumnos y sus notas, mostrarlos, calcular medias y consultar información concreta.

Idea clave: este proyecto te obliga a representar datos de forma más estructurada y a consultarlos con criterio.

Qué conceptos estás combinando

  • diccionarios
  • listas, si decides guardar varias notas
  • funciones
  • condicionales
  • bucles
  • validación de rangos
  • búsqueda por nombre
  • cálculo de medias

Situación

Imagina que quieres un programa sencillo para registrar alumnos y consultar sus notas. El usuario debería poder añadir alumnos, guardar sus notas y obtener información útil rápidamente.

Registrar

Guardar nombre y nota de un alumno.

Consultar

Ver todos los alumnos guardados.

Calcular

Obtener media general de las notas.

Buscar

Encontrar un alumno concreto por nombre.

Requisitos mínimos

  • debes usar un diccionario para guardar alumnos y notas
  • debes mostrar un menú dentro de un bucle while
  • debes usar funciones para organizar el programa
  • el programa debe permitir añadir un alumno con su nota
  • el programa debe permitir mostrar todos los alumnos
  • el programa debe permitir buscar un alumno por nombre
  • el programa debe calcular la media general
  • debe validar que la nota esté entre 0 y 10
  • debe permitir salir
Aquí ya importa mucho más cómo estructuras la información, no solo que el programa “funcione”.

Estructura de datos recomendada

La forma más sencilla de plantearlo en este nivel es un diccionario donde:

  • la clave sea el nombre del alumno
  • el valor sea su nota
alumnos = {
    "Ana": 8.5,
    "Luis": 6,
    "Marta": 9
}
Esta estructura encaja muy bien porque permite buscar por nombre de forma natural.

Decisión importante del proyecto

Aquí una de las decisiones clave no es solo programar el menú, sino elegir bien cómo representar la información. Si organizas mal los datos desde el principio, luego consultar, buscar o calcular medias será mucho más difícil.

Fases del proyecto

Construirlo por fases te ayudará a no bloquearte.

Fase 1

Crea el menú y haz que el programa pueda salir.

Fase 2

Añade la opción de guardar un alumno con su nota.

Fase 3

Muestra todos los alumnos con sus notas.

Fase 4

Implementa la búsqueda de un alumno por nombre.

Fase 5

Calcula la media general de las notas.

Fase 6

Mejora las validaciones y los mensajes.

Pistas opcionales

Pista 1

Usa un diccionario llamado alumnos.

Pista 2

Antes de guardar la nota, conviértela a número y valida que esté entre 0 y 10.

Pista 3

Puedes recorrer el diccionario con for nombre, nota in alumnos.items().

Pista 4

Para calcular la media, suma todas las notas y divide entre el número de alumnos.

Pista 5

Antes de calcular la media, comprueba que haya al menos un alumno guardado.

Pista 6

Para buscar por nombre, comprueba si la clave existe en el diccionario.

Errores frecuentes

Error 1

Guardar notas fuera del rango válido sin validarlas antes.

Error 2

Calcular la media sin comprobar antes si hay alumnos registrados.

Error 3

Buscar un alumno sin comprobar si la clave existe en el diccionario.

Error 4

Confundir una buena estructura de datos con una acumulación desordenada de variables sueltas.

Versión ampliada

Si terminas la versión principal, puedes añadir mejoras como estas:

  • mostrar aprobados y suspensos por separado
  • mostrar la nota más alta y la más baja
  • permitir modificar la nota de un alumno
  • permitir borrar un alumno
  • guardar varias notas por alumno usando listas

Qué deberías haber conseguido al terminar

Este proyecto marca un paso importante hacia programas más organizados.

Competencias que consolidan

  • usar diccionarios para representar datos estructurados
  • organizar un programa con funciones
  • recorrer y consultar información guardada
  • validar datos antes de usarlos
  • calcular resultados a partir de datos almacenados
