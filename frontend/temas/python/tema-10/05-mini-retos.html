10.5 · Mini retos: integrar manejo de errores y validaciones

Mini-reto 1 · Conversión segura

texto = input("Escribe un número: ")

try:
    numero = int(texto)
    print("Número correcto")
except ValueError:
    print("Entrada no válida")

Mini-reto 2 · División segura

try:
    a = 10
    b = 0
    print(a / b)
except ZeroDivisionError:
    print("No se puede dividir entre cero")

Mini-reto 3 · Abrir archivo

try:
    with open("datos.txt", "r") as archivo:
        print(archivo.read())
except FileNotFoundError:
    print("Archivo no encontrado")

Mini-reto 4 · Validación con raise

def validar_nota(nota):
    if nota < 0 or nota > 10:
        raise ValueError("Nota inválida")
    return nota

Mini-reto 5 · else y finally

try:
    numero = int("7")
except ValueError:
    print("Error")
else:
    print("Todo correcto")
finally:
    print("Fin del proceso")
