Tema 1 (Python) – Introducción al lenguaje

En este tema se explican con detalle los conceptos básicos de Python: qué es, cómo se ejecuta, variables, tipos de datos, operaciones y errores habituales. Es la base para todo lo que vendrá después.

1. Objetivos del tema

Al terminar este tema deberías ser capaz de:

  • Reconocer la estructura básica de un programa en Python.

No hace falta memorizar todo a la primera. La idea es que esta página sea una referencia a la que puedas volver cuando tengas dudas.

2. ¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Eso significa, en pocas palabras:

  • Alto nivel: está más cerca del lenguaje humano que del lenguaje máquina.
  • Interpretado: el código se ejecuta línea a línea, sin pasar por un proceso de compilación manual (como en C).
  • Propósito general: se puede usar para muchas cosas diferentes: desarrollo web, ciencia de datos, scripts, automatización, IA, etc.

Uno de los motivos principales por los que se usa tanto en educación es que su sintaxis es muy legible y sencilla. Un "Hola mundo" en Python se ve así: 

print("Hola mundo")

Esa sola línea ya es un programa completo en Python.

3. Cómo se ejecuta un programa en Python

Normalmente, para ejecutar un programa en Python se guardaría el código en un archivo con extensión .py (por ejemplo programa.py) y se ejecutaría con el intérprete de Python.

Python lee las instrucciones de arriba hacia abajo. En este ejemplo: 

x = 10
y = 2
print(x / y)
print("Fin del programa")

El intérprete hará lo siguiente:

  1. Crear una variable x con valor 10.
  2. Crear una variable y con valor 2.
  3. Calcular x / y (resultado: 5.0) y mostrarlo.
  4. Mostrar el texto Fin del programa.

Si en algún punto hay un error (por ejemplo, una variable mal escrita), Python se detiene y muestra un mensaje explicando qué ha pasado.

4. Comentarios e indentación

Dos características muy importantes de Python son los comentarios y la indentación.

4.1. Comentarios

Un comentario es una línea que Python ignora. Sirve para que el programador se deje notas a sí mismo o explique qué hace un bloque de código. En Python, los comentarios empiezan con #: 

# Esto es un comentario
print("Hola")  # Comentario al final de la línea

4.2. Indentación

La indentación es el espacio al principio de una línea (normalmente cuatro espacios). En otros lenguajes se usan llaves {} para indicar bloques; en Python se usa la indentación. Un ejemplo sencillo: 

edad = 18

if edad >= 18:
    print("Eres mayor de edad")
    print("Puedes entrar")
print("Fin")

Las dos líneas que están debajo del if y desplazadas hacia la derecha forman parte del bloque del if. Si cambiamos la indentación, cambiaremos el comportamiento del programa.

5. Variables en Python

Una variable es un "nombre" que apunta a un valor. La idea es poder reutilizar ese valor sin tener que escribirlo una y otra vez. En Python, la sintaxis básica es: 

nombre_variable = valor

Por ejemplo:

nombre = "Ana"
edad = 21
altura = 1.70
  • nombre es una variable de tipo cadena (str).
  • edad es un entero (int).
  • altura es un número con decimales (float).

En Python no se indica el tipo al declarar la variable, el intérprete lo deduce automáticamente en función del valor.

6. Tipos de datos básicos

Algunos tipos básicos que se usan constantemente en Python son:

  • int: números enteros, como 3, -10, 0.
  • float: números decimales, como 3.14, 0.5.
  • str: cadenas de texto, que van entre comillas ("Hola").
  • bool: valores lógicos True o False.
entero = 42           # int
decimal = 3.14        # float
texto = "Hola"        # str
activo = True         # bool

Más adelante verás otros tipos como listas, diccionarios, tuplas, etc., pero para empezar con los primeros ejercicios del Tema 1 estos son suficientes.

7. Operaciones aritméticas básicas

Con los tipos numéricos (int y float) se pueden realizar operaciones aritméticas estándar: 

a = 5
b = 3

print(a + b)   # Suma: 8
print(a - b)   # Resta: 2
print(a * b)   # Multiplicación: 15
print(a / b)   # División: 1.6666...
print(a // b)  # División entera: 1
print(a % b)   # Resto: 2
print(a ** b)  # Potencia: 5 elevado a 3 = 125

Muchas veces los ejercicios del tema 1 se centran en este tipo de operaciones: crear variables y mostrar resultados por pantalla.

8. Cadenas de texto (strings)

Una cadena (o string) es una secuencia de caracteres, normalmente escrita entre comillas dobles o simples: 

mensaje = "Hola mundo"
otro = 'Python es divertido'

Podemos "unir" cadenas de varias formas. Por ejemplo, usando comas en el print: 

nombre = "Mario"
print("Me llamo", nombre)

También se pueden usar f-strings (cadenas formateadas) a partir de Python 3.6: 

nombre = "Ana"
edad = 20
print(f"Me llamo {nombre} y tengo {edad} años")

En el Tema 1 nos quedaremos con la forma sencilla usando comas, que es suficiente para empezar.

9. Errores habituales al empezar con Python

Es totalmente normal que al aprender un lenguaje aparezcan errores. De hecho, aprender a leer e interpretar errores es parte importante del aprendizaje.

9.1. Variable no definida

print(x)  # x no existe todavía

Esto generará un error del tipo: 

NameError: name 'x' is not defined

9.2. Problemas con las comillas

print("Hola)   # Falta una comilla

Aquí aparecerá un error de sintaxis (SyntaxError) porque la cadena no está bien cerrada.

9.3. División entre cero

x = 5
y = 0
print(x / y)   # ZeroDivisionError

Python no permite dividir entre cero y mostrará un ZeroDivisionError.

En el playground de tu aplicación, estos errores aparecerán en la parte de stderr. Es recomendable leerlos con calma e intentar entender qué significan.
