9.2 · Leer archivos: recuperar información guardada

Lectura completa

archivo = open("datos.txt", "r")
contenido = archivo.read()
archivo.close()
print(contenido)

Leer línea a línea

archivo = open("datos.txt", "r")
for linea in archivo:
    print(linea)
archivo.close()

Muy útil cuando el archivo tiene varias líneas.

Usar with

with open("datos.txt", "r") as archivo:
    contenido = archivo.read()
    print(contenido)
with es la forma recomendada porque cierra el archivo automáticamente.

⚠️ Error típico: archivo no encontrado

Si la ruta o el nombre del archivo es incorrecto, el programa falla.
