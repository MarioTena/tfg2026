9.1 · Qué es un archivo: guardar datos fuera del programa

¿Qué problema resuelven los archivos?

Hasta ahora, los datos vivían en variables mientras el programa se ejecutaba. Pero cuando el programa termina, esas variables desaparecen.

Un archivo permite guardar información de forma persistente para reutilizarla más tarde.

Ejemplos reales

  • guardar notas en un archivo de texto
  • leer configuraciones
  • registrar logs
  • guardar resultados de un programa

La idea general

Para trabajar con archivos, normalmente sigues este patrón:

  1. abrir el archivo
  2. leer o escribir
  3. cerrarlo

Función open()

archivo = open("datos.txt", "r")

Esto abre el archivo en modo lectura.

Modos habituales

  • "r" leer
  • "w" escribir sobrescribiendo
  • "a" añadir al final
